# 网络爬虫

## 网络爬虫

网页地址主要由几部分组成

* `http/https` 协议
* `www.baidu.com` 主机
* `80/443` 端口
* `s` 路径
* `wd=` 参数
* `#` 锚点



### urllib

urllib 库是 Python 默认的库，可以获得网页内容。例如

```python
import urllib.request

# 打开网页
url = "https://zjuers.com/"
response = urllib.request.urlopen(url)

# 读取网页内容（二进制）然后解码
content = response.read().decode('utf-8')

print(content)
```



#### 打开网页

urlopen 方法返回 http.client.HTTPResponse 类。具有如下常用方法

| 方法         | 作用                        |
| ------------ | --------------------------- |
| read(n)      | 读取 n 个字节               |
| readline()   | 读取一行                    |
| readlines()  | 每次读一行，直到读完        |
| getcode()    | 获得状态码，若为 200 则正常 |
| getheaders() | 获得状态信息                |



#### 下载资源

urlretrieve 方法可以下载链接资源到指定文件。例如

```python
url = "https://zjuers.com/"
urllib.request.urlretrieve(url, 'zjuers.html')
```



#### 定制请求对象

某些网站具有 UA(User-Agent) 反爬虫机制，此时就需要定制对象。首先在指定网页

1. 检查元素（需要刷新）
2. 在 Network 左侧 Name 栏中找到网页
3. 在右侧 Headers 栏找到 User-Agent

![[网络爬虫.assets/image-20230619220715140.png]]

然后添加用户代理

```python
import urllib.request

# 打开网页
url = "https://www.baidu.com/"

# 配置用户代理
headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/114.0.0.0 Safari/537.36'
}

# 定制 Request 对象
request = urllib.request.Request(url=url, headers=headers)
response = urllib.request.urlopen(request)

content = response.read().decode('utf-8')

print(content)
```

当然这种方法只能处理 UA 反爬的网页。



#### Unicode 编码

进行爬虫时，可能需要使用中文进行搜索，例如

```python
url = "https://www.baidu.com/s?wd=周杰伦"
```

但是显然不能识别这样的网址。需要使用 quote 方法将汉字转换为 Unicode 编码

```python
import urllib.parse

name = urllib.parse.quote('周杰伦')
url = "https://www.baidu.com/s?wd=" + name
```

或者使用 urlencode 方法将字典数据转换为 Unicode 编码

```python
data = {
    'wd': '周杰伦',
    'sex': '男',
    'location': '台湾'
}

# 得到 wd=%E5%91%A8%E6%9D%B0%E4%BC%A6&sex=%E7%94%B7&location=%E5%8F%B0%E6%B9%BE
ndata = urllib.parse.urlencode(data)
```

这种方法可以快速将搜索信息转换为一组编码信息。



#### POST 请求

POST 是一种网页请求，可以**以表单形式上传数据**，除查询信息外，还可以修改信息。



以百度翻译为例，输入英文单词，在检查元素中寻找用于翻译的请求。可以看到 sug 请求出现次数最多，点击查看

* Request URL 即接口为 `https://fanyi.baidu.com/sug`
* Request Method 为 POST，说明它是 POST 请求

可以在 Payload 中找到 Form Data 为 `kw: spider`，这就是查询的数据。

![[网络爬虫.assets/image-20230619223533535.png]]



然后我们就可以通过这个 url 接口进行翻译

```python
import urllib.request
import urllib.parse
import json

# 翻译接口
url = "https://fanyi.baidu.com/sug"

# 配置用户代理
headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/114.0.0.0 Safari/537.36'
}

# 需要翻译的数据
data = {
    'kw': 'spider'
}

# post 请求的参数需要编码
data = urllib.parse.urlencode(data).encode('utf-8')

# 定制 Request 对象，指定接口、数据和用户代理
request = urllib.request.Request(url=url, data=data, headers=headers)
response = urllib.request.urlopen(request)

content = response.read().decode('utf-8')

# 反序列化为对象
content = json.loads(content)

print(content)
```

#### GET 请求

GET 请求是最常见的方式，一般用于获取或者查询资源信息，也是大多数网站使用的方式，响应速度快。



在豆瓣电影排行榜中选择“剧情”类，然后刷新检查所有请求，可以看到这个返回大量 json 数据的请求应该就是我们要爬取的数据接口

* Request URL: `https://movie.douban.com/j/chart/top_list?type=11&interval_id=100%3A90&action=&start=0&limit=20`
* Request Method: GET

使用 GET 请求获得数据。

![[网络爬虫.assets/image-20230620103442826.png]]



使用 GET 方式获得数据，不需要上传 data 数据

```python
import urllib.request
import urllib.parse
import json

# 电影信息接口
url = "https://movie.douban.com/j/chart/top_list?type=11&interval_id=100%3A90&action=&start=0&limit=20"

headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/114.0.0.0 Safari/537.36'
}

request = urllib.request.Request(url=url, headers=headers)
response = urllib.request.urlopen(request)

content = response.read().decode('utf-8')

fp = open('double.json', 'w', encoding='utf-8')
fp.write(content)
fp.close()
```



通过查看每一页的数据接口可以得到

```shell
https://movie.douban.com/j/chart/top_list?type=11&interval_id=100%3A90&action=&start=0&limit=20
https://movie.douban.com/j/chart/top_list?type=11&interval_id=100%3A90&action=&start=20&limit=20
https://movie.douban.com/j/chart/top_list?type=11&interval_id=100%3A90&action=&start=40&limit=20
```

观察接口规律，每一页数据的 start 参数相差 20，limit 应该是限制每次获得的条数。于是就可以进一步获得指定页的数据

```python
import urllib.request
import urllib.parse
import json


# 创建指定页的 request 链接
def create_request(page, limit=20):
    base_url = 'https://movie.douban.com/j/chart/top_list?type=11&interval_id=100%3A90&action=&'

    headers = {
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/114.0.0.0 Safari/537.36'
    }

    data = {
        'start': (page - 1) * limit,
        'limit': limit
    }

    data = urllib.parse.urlencode(data)
    url = base_url + data

    request = urllib.request.Request(url=url, headers=headers)
    return request


# 获得内容
def get_content(request):
    response = urllib.request.urlopen(request)
    content = response.read().decode('utf-8')
    return content


# 指定下载文件
def download(page, content):
    with open('double_' + str(page) + '.json', 'w', encoding='utf-8') as fp:
        fp.write(content)


# 程序入口
if __name__ == '__main__':
    start = int(input('起始页面：'))
    end = int(input('结束页面：'))

    for p in range(start, end + 1):
        request = create_request(p)
        content = get_content(request)
        download(p, content)

```

#### Handler 处理器

Handler 可以定制更高级的请求头，例如处理**动态 cookie 和代理不能使用请求对象**的定制。



原先使用 urllib.request.urlopen 爬取的操作可以分解成三部分

```python
import urllib.request
import urllib.parse
import json

url = "https://movie.douban.com/j/chart/top_list?type=11&interval_id=100%3A90&action=&start=0&limit=20"

headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/114.0.0.0 Safari/537.36'
}

request = urllib.request.Request(url=url, headers=headers)

handler = urllib.request.HTTPHandler()          # 获得 handler 对象
opener = urllib.request.build_opener(handler)   # 获取 opener 对象
response = opener.open(request)                 # 调用 opener 方法爬取

content = response.read().decode('utf-8')

print(content)
```



#### 代理 IP

使用代理可以避免同一 IP 频繁访问网页可能导致 IP 被封的问题。使用**快代理**提供的 IP 和对应的端口 PORT 建立代理 handler 对象

```python
import urllib.request
import urllib.parse
import random
import json

url = "https://movie.douban.com/j/chart/top_list?type=11&interval_id=100%3A90&action=&start=0&limit=20"

headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/114.0.0.0 Safari/537.36'
}

# 建立代理池 IP : PORT
proxies_pool = [
    {'http': '60.167.21.14:1133'},
    {'http': '222.74.73.202:42055'}
]

# 随机选择 IP
proxy = random.choice(proxies_pool)

request = urllib.request.Request(url=url, headers=headers)

handler = urllib.request.ProxyHandler(proxies=proxy)      	# 创建代理 handler 对象
opener = urllib.request.build_opener(handler)               # 获取 opener 对象
response = opener.open(request)                             # 调用 opener 方法

content = response.read().decode('utf-8')

print(content)
```

##### 路径查询

```python
from lxml import etree

tree = etree.parse('test.html')

li_list1 = tree.xpath('//body/ul/li')   # 查找 body 下的 ul 下的所有 li 子标签
li_list2 = tree.xpath('//body//li')     # 查找 body 下的所有 li 子/孙标签
li_list3 = tree.xpath('//body//li[1]')	# 查找 body 下的第一个 li 子/孙标签

print(len(li_list1))
print(len(li_list2))
print(len(li_list3))
```

注意 `[]` 的索引从 1 开始。



可以对 xpath 返回的对象再次使用 xpath，例如

```python
li = li_list1[2]
a = li.xpath('./a/text()')
```

注意使用 `.` 表示相对路径。



##### 谓词查询

```python

tree = etree.parse('test.html')

li_list4 = tree.xpath('//body//li[@id="beijing"]/@class')   # 查找 id 属性为 beijing 的标签的 class 属性

print(li_list2)
print(li_list3)
print(li_list4)
```

##### 站长素材

我们进入站长素材网站，搜索风景图片。解析网页源码获得图片的链接进行下载。首先找到图片所在的位置，然后使用 xpath 插件进行路径查询，验证语法正确性

![image-20230701110758608](image-20230701110758608.png)



然而需要注意两个问题

* 通过 urllib 下载的类名 `item masonry-brick` 会变成 item，需要修改对应的 xpath 查询内容
* 由于网页使用“懒加载”，未显示的部分图片不会进行加载，对应的标签中没有 src 属性。我们需要使用 data-original 属性获得图片的原始链接

批量下载指定页图片的代码框架如下

```python
import urllib.request
from lxml import etree


# 创建指定页的 request 链接
def create_request(page):
    # 第一页的格式不一样
    if page == 1:
        url = 'https://sc.chinaz.com/tupian/fengjingtupian.html'
    else:
        url = 'https://sc.chinaz.com/tupian/fengjingtupian_' + \
            str(page) + '.html'

    headers = {
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/114.0.0.0 Safari/537.36'
    }

    request = urllib.request.Request(url=url, headers=headers)
    return request


# 获得内容
def get_content(request):
    response = urllib.request.urlopen(request)
    content = response.read().decode('utf-8')
    return content


# 指定下载文件
def download(content):
    tree = etree.HTML(content)

    name_list = tree.xpath(
        '//div[@class="container"]/div/div[@class="item"]/img/@alt')
    src_list = tree.xpath(
        '//div[@class="container"]/div/div[@class="item"]/img/@data-original')

    for i in range(len(name_list)):
        name = name_list[i]
        src = src_list[i]
        
        # 补充 https: 前缀，去掉 _s 后缀，下载原始高清图片
        url = 'https:' + src.replace('_s', '')
        urllib.request.urlretrieve(url=url, filename='./sc/' + name + '.jpg')


# 程序入口
if __name__ == '__main__':
    start = int(input('起始页面：'))
    end = int(input('结束页面：'))

    for p in range(start, end + 1):
        request = create_request(p)
        content = get_content(request)
        download(content)
```

BeautifulSoup 简称为 bs4，它用于解析和提取数据。它的优点是接口设计人性化，使用方便，但是缺点是效率比 lxml 低。通常大规模网页爬取使用 lxml，而简单网页爬取使用 bs4 即可。

#### Chrome handless

Chrome handless 是 Google 新增的一种模式，可以在不打开 UI 界面的情况下使用 Chrome 浏览器。优势在于不需要进行页面的渲染，因此可以提高爬取效率。



我们用如下代码封装浏览器对象

```python
from selenium import webdriver
from selenium.webdriver.chrome.service import Service
from selenium.webdriver.chrome.options import Options


# 浏览器封装
def share_browser(hideWindow=True):
    # 配置对象
    options = webdriver.ChromeOptions()
    options.add_experimental_option('detach', True)     # 不自动关闭浏览器
    
    if hideWindow:
        options.add_argument('--headless')      # 设置无窗口模式
        options.add_argument('--disable-gpu')   # 禁用 gpu 加速

    # 创建浏览器
    service = Service('chromedriver.exe')
    browser = webdriver.Chrome(service=service, options=options)
    return browser


# 创建浏览器
browser = share_browser()

url = 'https://www.baidu.com'
browser.get(url)

# 保存截图
browser.save_screenshot('baidu.png')
```

最终的爬虫代码如下

```python
import requests
from bs4 import BeautifulSoup

url = "https://so.gushiwen.cn/user/login.aspx?from=http://so.gushiwen.cn/user/collect.aspx"

headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/114.0.0.0 Safari/537.36'
}

# 获得网页源码
response = requests.get(url=url, headers=headers)
content = response.text

# 解析获得参数
soup = BeautifulSoup(content, 'lxml')

viewstate = soup.select('#__VIEWSTATE')[0].attrs.get('value')
viewstategenerator = soup.select('#__VIEWSTATEGENERATOR')[0].attrs.get('value')

# 获得验证码图片
code = soup.select('#imgCode')[0].attrs.get('src')
code_url = "https://so.gushiwen.cn" + code

# 获得 session 请求对象
session = requests.session()
response_code = session.get(code_url)

# 获得 url 的二进制数据，写入一张图片文件
content_code = response_code.content
with open('code.jpg', 'wb') as fp:
    fp.write(content_code)

# 查看图片，输入验证码
code_name = input('输入验证码：')

# 提供登录数据
data = {
    '__VIEWSTATE': viewstate,
    '__VIEWSTATEGENERATOR': viewstategenerator,
    'from': 'http://so.gushiwen.cn/user/collect.aspx',
    'email': '814083398@qq.com',
    'pwd': 'xingyifan666',
    'code': code_name,
    'denglu': '登录'
}

# 使用 session 提交 POST 请求
response_post = session.post(url=url, headers=headers, data=data)

# 获得网页源码
content_post = response_post.text
with open('gushiwen.html', 'w', encoding='utf-8') as fp:
    fp.write(content_post)
```

```python
import scrapy
from project.items import ProjectItem


class DangSpider(scrapy.Spider):
    name = "dang"

    # 注意多页下载，必须提供基本域名作为允许访问的域名
    allowed_domains = ["category.dangdang.com"]
    start_urls = ["https://category.dangdang.com/cp01.01.01.00.00.00.html"]

    # 为多页爬取提供基本网址
    base_url = 'https://category.dangdang.com/pg'
    page = 1

    def parse(self, response):
        # 图片路径 //ul[@class="bigimg"]/li//img/@src
        # 描述路径 //ul[@class="bigimg"]/li//img/@alt
        # 价格路径 //ul[@class="bigimg"]/li//p[@class="price"]/span[1]/text()

        # 获得 li 对象，可以再次调用 xpath
        li_list = response.xpath('//ul[@class="bigimg"]/li')

        # 需要注意图片懒加载的问题，只有第一张图片使用 src 属性，其它都通过 data-original 属性获得
        for li in li_list:
            src = li.xpath('.//img/@data-original').extract_first()
            name = li.xpath('.//img/@alt').extract_first()
            price = li.xpath(
                './/p[@class="price"]/span[1]/text()').extract_first()

            if not src:
                src = li.xpath('//img/@src').extract_first()

            # 创建数据对象，交给 pipelines
            book = ProjectItem(src=src, name=name, price=price)
            yield book

        # 进行多页爬取，重复调用方法即可
        if self.page < 5:
            self.page = self.page + 1

            # http://category.dangdang.com/pg2-cp01.01.01.00.00.00.html
            url = self.base_url + str(self.page) + '-cp01.01.01.00.00.00.html'

            # 通过 scrapy 的 Request 方法作为 get 请求，指定回调函数为 parse 来重新调用 parse 函数
            yield scrapy.Request(url=url, callback=self.parse)

```

```python
    name = "movie"
    allowed_domains = ["www.ygdy8.net"]
```

## 正则表达式

正则表达式通过一个字符序列表示满足某种逻辑条件的字符串，主要用于字符串模式匹配或字符串匹配。



需要导入 re 库

```python
import re
```

可以直接匹配需要的字符串格式

```python
re.findall('x', '122xxx4xx')			# 匹配所有字符串
re.search('abc', 'hello abc and you')	# 搜索第一个字符串
```



### 基本语法

使用正则表达式进行模糊匹配，常用的匹配格式包括

* 字符限定
* 数量限定
* 分组
* 定位



#### 字符限定符

| 字符       | 作用                             |
| -------- | ------------------------------ |
| `[abc`]  | [] 中给出一个字符集合，可以匹配其中的**任意一个**字符 |
| `[^abc]` | 与该集合外的任意一个字符匹配                 |
| `[a-z]`  | 匹配 a-z 范围的一个字符匹配               |
| `[^a-z]` | 与该集合外的一个字符匹配                   |
| `.`      | 匹配除换行符外的**任意一个**字符             |



使用 `\` 转义字符来匹配格式

| 转义符  | 作用                    |
| ---- | --------------------- |
| `\d` | 匹配一个数字字符              |
| `\D` | 匹配一个非数字字符             |
| `\w` | 匹配一个单词（包括数字、字母、下划线）   |
| `\W` | 匹配一个非单词字符             |
| `\s` | 匹配一个不可见字符（空格、制表符、换行符） |
| `\S` | 匹配一个可见字符              |



使用**三位数字**表示匹配八进制数对应的 ASCII 字符。例如

```python
re.search('(FishC)\\060', 'FishCFishC0')
```

这里 `\060` 被解析为八进制数，对应的 ASCII 字符是 `0`，匹配 `FishC0` 。



使用 `x` 开头表示匹配十六进制，例如

```python
re.search('(FishC)\\x61', 'FishCFishCa')
```

这里 `\x61` 对应字符 `a`，匹配 `FishCa` 。



#### 数量限定符

数量限定符指定模式的重复次数。例如

| 字符                | 作用                                                 |
| ----------------- | -------------------------------------------------- |
| `*`               | 前面的字符进行任意（0到多次）次匹配                                 |
| `+`               | 前面的字符进行一次或多次匹配                                     |
| `?`               | 前面的字符进行 0 次或一次匹配                                   |
| `{n}`             | 前面的字符进行 n 次（非负）匹配                                  |
| `{n,}`            | 前面的字符进行至少n次（非负）匹配                                  |
| `{n,m}`           | 前面的字符进行至少 n 次，最多 m 次匹配                             |
| `*?, +?, ??, {}?` | `?` 跟在任何数量限定元字符后表示**非贪婪匹配**，即尽可能少地匹配（默认情况下尽可能多地匹配） |

为了避免错误，不要在 `{}` 中随意添加空格。



例如匹配任意数量的字符

```python
re.search('.+', 'FishCFishCa')
```

匹配 0 次或一次任意字符

```python
re.search('.?', 'FishCFishCa')
```



#### 分组

使用 `()` 包围的表达式被视为一个整体，前面的数量限定将会作用于这个整体。例如

```python
re.search('(\d\d\d)\.(\d\d\d)', '123.856')
```

可以使用 `|` 表示或匹配

```python
re.search('([a-z])|(\d\d\d\d)', 'a1234.856')
```

它匹配一个字母或者 4 个数字。



可以使用 `\` 来引用组，例如

```python
re.search('(FishC)\\1', 'FishCFishC')
```

这里 `\1` 就引用了第一个组，等价于

```python
re.search('(FishC)(FishC)', 'FishCFishC')
```



#### 定位

可以确定模式匹配的位置

| 字符   | 作用                   |
| ---- | -------------------- |
| `^`  | 所在的组必须出现在开头才可以匹配     |
| `$`  | 所在的组必须出现在末尾才可以匹配     |
| `\b` | 匹配一个单词边界（单词和空格之间的位置） |
| `\B` | 匹配一个非单词边界            |

例如匹配开头字符串

```python
re.search('^(123)', '1234')		# 可以匹配
re.search('^(123)', '51234')	# 不能匹配
```



### 常用方法

可用的正则表达式匹配方法有很多，我们只介绍其中常用的部分。

| 方法    | 作用                                                         |
| ------- | ------------------------------------------------------------ |
| compile | 匹配任何可选标记编译正则表达式的模式，返回正则表达式对象     |
| search  | 使用可选标记搜索字符串中第一次匹配正则表达式的对象，匹配成功返回匹配对象；否则返回None |
| match   | 使用带有可选标记的正则表达式的模式匹配字符串，从第一个字符开始匹配，匹配成功返回匹配对象；否则返回None |
| findall | 查找字符串中所有匹配的子字符串，返回一个字符串匹配列表。如果匹配模式中有分组，则会返回元组列表 |
| sub     | 替代匹配的子字符串，返回替代的结果                           |



#### findall

这是最常用的匹配方法，findall 方法会匹配字符串中所有符合模式的子字符串，并分组返回。例如

```python
import re

s = "Hello.123.world.456"
res = re.findall('[0-9]+', s)
print(res)
```

将会返回 `['123', '456']` 列表。



如果指定分组，就会返回元组的列表，例如

```python
import re

s = "Hello-123-world-456"
res = re.findall('([a-zA-z]+)-([0-9]+)', s)
print(res)
```

将会匹配 `Hello-123` 和 `world-456`，但是由于用 `()` 进行了分组，则只会返回分组的列表

```shell
[('Hello', '123'), ('world', '456')]
```

可以看到中间的 `-` 没有返回。



#### sub

如果需要替换掉匹配的部分，使用 sub 方法。例如用指定的字符串进行替换

```python
import re

s = "Hello-123-world-456"
res = re.sub('([a-zA-z]+)-([0-9]+)', 'Hello', s)
print(res)
```

将会得到 `Hello-Hello` 字符串。



也可以传入替代函数

```python
import re

def dashrepl(matchobj):
    print("group0:", matchobj.group(0))
    print("group1:", matchobj.group(1))
    print("group2:", matchobj.group(2))
    return str(matchobj.group(0)).replace('-', '.')

s = "Hello-123-world-456"
res = re.sub('([a-zA-z]+)-([0-9]+)', dashrepl, s)
print(res)
```

注意 dashrepl 函数的参数，它的 group 方法得到匹配的组

* 0 是整个匹配结果，例如 `Hello-123`
* 1 是第一个组，即 `Hello`
* 2 是第二个组，即 `123`

以此类推可以得到匹配结果中的每个组。函数的返回值会**替换掉整个匹配结果**。



通常 `.` 不能匹配 `\n`，可以设置 flags 参数

```python
re.sub('..\$\$[^\$]+\$\$..', dashrepl, s, flags=re.DOTALL)
```

DOTALL 表示 `.` 可以匹配任何字符。

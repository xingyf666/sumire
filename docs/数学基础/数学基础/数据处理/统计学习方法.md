# 统计学习方法

## 统计学习概论

### 学习框架

#### 监督学习

监督学习 supervised learning 从标注数据集中学习预测模型。将输入与输出的所有可能性的集合分别称为输入空间 input space 和输出空间 output space 。监督学习将输入与输出看作是随机变量的取值，将输入随机变量记为 $X$，输出随机变量记为 $Y$，它们遵循联合概率分布 $P(X,Y)$ 。监督学习的模型包括

- 概率模型：条件概率分布 $P(Y| X)$
- 非概率模型：决策函数 $Y=f(X)$

学习系统从训练数据 training data 中学习模型，对测试数据 test data 进行预测。预测模型的集合称为假设空间 hypothesis space 。若输入为 $x_{N+1}$，则输出为
$$
\begin{aligned}
y_{N+1} &= \arg\max_{y}\hat{P}(y|x_{{N+1}})\\
y_{N+1} &= \hat{f}(x_{N+1})
\end{aligned}
$$
假设训练数据与测试输出依联合概率分布 $P(X,Y)$ 独立同分布产生。

#### 无监督学习

无监督学习 unsupervised learning 从无标注数据中学习预测模型。本质上是学习数据中的统计规律或潜在结构。设 $\mathcal{X}$ 为输入空间，$\mathcal{Z}$ 为隐式结构空间。要学习的模型可表示为

- $z=g(x)$
- $P(z|x)$
- $P(x|z)$

无监督学习从假设空间中选择给定评价标准下的最优模型。

#### 强化学习

强化学习 reinforcement learning 要求系统在与环境的连续互动中学习最优行为策略。假设系统与环境的互动基于马尔可夫决策过程 Markov decision process，系统能得到的是与环境互动后的数据序列。随机过程由五元组 $(S,A,P,r,\gamma)$ 组成

- $S$ 有限状态 state 的集合
- $A$ 有限动作 action 的集合
- $P$ 状态转移概率 transition probability 函数 $P(s'|s,a)=P(s_{t+1}=s'|s_{t}=s,a_{t}=a)$，其中 $s,a,t$ 表示状态、动作和时间
- $r$ 奖励函数 reward function $r(s,a)=E(r_{t+1}|s_{t}=s,a_{t}=a)$
- $\gamma$ 衰减系数 discount factor $\gamma\in[0,1]$

策略 $\pi$ 定义为 $a=f(s)$ 或 $P(a|s)$，它说明在不同状态下给出的动作。价值函数 value function 或状态价值函数 state value function 定义为策略 $\pi$ 从某个状态 $s$ 开始积累奖励的数学期望
$$
v_{\pi}(s) = E_{\pi}\left[ \sum_{i=0}^\infty\gamma^ir_{t+i}\bigg|s_{t}=s \right]
$$
动作价值函数 action value function 定义为策略 $\pi$ 从某个状态 $s$ 和动作 $a$ 开始积累奖励的数学期望
$$
q_{\pi}(s,a) = E_{\pi}\left[ \sum_{i=0}^\infty\gamma^ir_{t+i}\bigg|s_{t}=s,a_{t}=a \right]
$$
强化学习的目标是从所有可能的策略中选出价值函数最大的策略 $\pi^*$ 。

### 学习技巧
#### 贝叶斯学习

贝叶斯学习找到使后验概率 $P(\theta|D)$ 最大的模型参数 $\theta$，其中 $D$ 表示数据。根据贝叶斯定理
$$
P(\theta|D) = \frac{P(\theta)P(D|\theta)}{P(D)}
$$
其中 $P(\theta)$ 是先验概率，$P(D|\theta)$ 是似然函数。预测时计算后验概率分布的期望
$$
P(x|D) = \int P(x|\theta,D)P(\theta|D)d\theta
$$
其中 $x$ 就是使 $P(x|D)$ 最大的新样本。

>[!note]
>因为 $P(D)$ 固定，而通常认为 $\theta$ 均匀分布，因此可以通过贝叶斯定理将极大似然估计 $\hat{\theta}=\arg\max_{\theta}P(D|\theta)$ 与贝叶斯估计直接关联。



#### 核方法

核方法使用核函数表示核学习非线性模型，主要用于将线性模型扩展到非线性模型。它不直接定义输入空间（低维）到特征空间（高维）的映射，而是通过特征空间中的内积定义核函数，即
$$
K(x_{1},x_{2}) = \langle \varphi(x_{1}),\varphi(x_{2})\rangle
$$
将输入数据投射到高维特征空间中分析性质。



### 泛化能力

#### 泛化误差

学习方法的泛化能力 generalization ability 是指该方法学习到的模型对位置数据的预测能力。例如使用交叉验证防止过拟合、添加正则项限制模型的复杂度。定义泛化误差
$$
R_{exp}(\hat{f}) = E_{P}[L(Y,\hat{f}(X))] = \int_{\mathcal{X}\times \mathcal{Y}}L(y,\hat{f}(x))P(x,y)dxdy
$$
其中 $\hat{f}$ 是学习到的模型，$L$ 是损失函数。泛化误差衡量模型的期望风险。



#### 泛化误差上界

考虑二分类问题。已知训练集 $T=\{(x_{i},y_{i})\}_{i=1}^N$，它从联合概率分布 $P(X,Y)$ 独立同分布获得，且 $X\in \mathbb{R}^n,Y\in\{-1,+1\}$，假设空间为 $\mathcal{F}=\{f_{i}\}_{i=1}^d$，则
$$
\begin{aligned}
R(f) &= E[L(Y,f(X))]\\
\hat{R}(f) &= \frac{1}{N} \sum_{i=1}^NL(y_{i}f(x_{i}))
\end{aligned}
$$
分别表示期望风险和经验风险。经验风险最小化函数为
$$
f_{N} = \arg\min_{f\in \mathcal{F}}\hat{R}(f)
$$
我们关系 $f_{N}$ 的泛化能力
$$
R(f_{N}) = E[L(Y,f_{N}(X))]
$$
则有

>[!theorem] 泛化误差上界
>对二分类问题，若假设空间 $|\mathcal{F}|=d$ 有限，对任意 $f\in \mathcal{F}$，至少以概率 $1-\delta,0<\delta<1$ 成立
>$$
>R(f) \leq \hat{R}(f) + \epsilon(d,N,\delta)
>$$
>其中
> $$
>\epsilon(d,N,\delta) = \sqrt{ \frac{1}{2N}\left( \log d+\log \frac{1}{\delta} \right) }
>$$

使用 [[数据科学的数学基础#Chernoff-Hoeffding Inequality]] 可以证明此上界。

# Blender

## 基本介绍

### 观察视图

打开后会看到这样一个界面：

* 左侧是建模操作；
* 下方是时间轴；
* 右侧是场景信息；

通过鼠标中键转动视角；Shift + 鼠标中键平移视角

> 由于 utools 占用了鼠标中键，因此通过拖动右上角的小坐标轴以及“手部”图标实现转动视角和平移视角。

![[Blender.assets/image-20240205222400563.png]]



### 物体操作

左侧中间的 4 个图标可以调整物体。鼠标移动上去就可以看到快捷键

![[Blender.assets/image-20240205224435750.png]]



#### 调整模式

第一个图标是“调整”图标，它用于选中物体和移动物体等操作。里面还有

* 框选：可以一次选择多个物体
* 刷选：选择刷到的物体
* 套索选择：选择套索范围的物体



#### 游标模式

第二个图标是“游标”图标，它是画面中的十字准心。新建的物体将会出现在游标位置。处于游标模式下，可以移动游标位置。

* 在调整模式下，可以 Shift + 右键移动游标；
* 使用 Shift + c 恢复游标位置到原点；
* 使用 Shift + s 调出游标操作，移动选中项到游标或者移动游标到选中项等；

![[Blender.assets/image-20240206010408957.png]]



#### 物体原点

物体中的黄色点就是物体的原点，一般的变换都是围绕原点进行。当我们需要进行特殊的操作，例如将立方体的底部放在坐标原点，这时候就需要移动物体原点的位置。



在右上角选择“仅影响原点”，

* 这时候进行 g 平移操作就会只移动物体原点，切换到 y 视图，按住 Ctrl 键移动原点吸附到立方体底部，然后取消勾选“仅影响原点”。最后 Alt + g 让物体原点回到坐标原点，那么立方体底部就移动到了原点位置。
* 如果需要将原点移动到坐标原点，直接 Alt + g 即可，然后取消勾选“仅影响原点”。

或者直接右键选择“设置原点”来移动原点位置。

![[Blender.assets/image-20240206010820267.png]]



#### 轴心点

默认轴心点事物体原点，即质心点。当选择多个物体时，

* 边界框中心：用一个最小的方框框住所有物体，取这个方框的中心
* 3D 游标：围绕游标变换
* 质心点：所有物体的质心
* 各自的原点：围绕各自原点变换
* 活动元素：围绕最后选中的物体变换

![[Blender.assets/image-20240206012131661.png]]



#### 局部坐标系

在操作时需要区分局部坐标系和全局坐标系。在全局坐标系下，按 g + x 会沿着全局坐标的 x 轴平移；按 g + x + x 则切换到局部坐标的 x 轴进行平移；其余坐标轴同理。反之，在局部坐标轴下，按 g + x 会沿着局部坐标的 x 轴平移；按 g + x + x 则切换到全局坐标的 x 轴进行平移；其余坐标轴同理。

![[Blender.assets/image-20240206012717875.png]]



#### 应用变换

这是一个非常重要的部分！！！**尽量在编辑模式下进行缩放操作！在物体模式下进行的缩放操作，最好用 Ctrl + a 应用缩放。**



按 n 键查看条目，会发现“变换”部分给出了对物体施加的变换。这意味着**这些变换没有修改物体本身的数据，是在物体数据的基础上进行变换**。而使用 Ctrl + a 应用变换后，**变换将会应用到物体数据，此时物体本身发生了变化，而所有变换都被清空**。

![[Blender.assets/image-20240224010209408.png]]

## 网格建模

### 基本控制

为了方便切换，我们在“偏好设置”中勾选“拖动时的饼菜单”，这样可以长按 Tab 键移动鼠标调出切换模式的菜单。

![[Blender.assets/image-20240208225058143.png]]



#### 切换模式

进入编辑模式后，可以通过键盘上方（不是数字键）的 1/2/3 切换点/线/面编辑。或者通过菜单图标切换，图标设置可以同时开启多个选择，按住 Shift 键可以全部选中 3 种模式，这样可以同时选择点、线、面。

![[Blender.assets/image-20240208225403252.png]]



#### 选择技巧

在选择时，可以长按

* Shift 键加选
* Ctrl 键减选；配合数字键盘，Ctrl + + 可以扩大选区；Ctrl + - 可以减小选区 

一些常用的选择方法：

* 选择一个点/线/面，按住 Ctrl 键再选择另一个点/线/面，会自动选择最短路径上的点/线/面；
* 选择一个点/线/面，按 l 键选择与当前选中元素相连的所有元素（需要鼠标移动到物体上）；或者直接按 Ctrl + l 选中；
* 选择一个点/线/面，
    * 按住 Alt 键，鼠标轻微移动到一个方向上双击，就可以选中这个方向上的所有元素；
    * 按住 Ctrl + Alt 键，鼠标轻微移动到一个方向上双击，就可以选中垂直于这个方向上的所有元素；
* 还可以通过“选择”菜单，进行随机选择。

![[Blender.assets/image-20240208230802877.png]]

![[Blender.assets/image-20240208230854264.png]]



对于分段的连续边，使用 Shift + Alt 只会选择其中一段上的边，这时候可以使用“选择锐边”选取连续角度不超过指定度数的一系列边

![[Blender.assets/image-20240223220356239.png]]



要选择背面的点，需要开启“透视模式”。

![[Blender.assets/image-20240208231236025.png]]

然而，Alt + z 被英伟达占用了。作为替代，可以用 z 进入线框渲染模式。按住 z 再拖动鼠标可以调出切换面板。



#### 删除技巧

选中元素，按 x 键选择删除内容。对于边，可以删除边或者删除边的顶点；其余元素类似。使用“融并”可以在删除元素后自动连接其余元素，不会出现破洞。

![[Blender.assets/image-20240208231741126.png]]



#### 法向和法线

勾选“面朝向”可以查看面的正反，蓝色的是正面，红色的是反面。

![[Blender.assets/image-20240208231959694.png]]

在编辑模式下，可以选择“法向”，展示点法向或者面法向等信息。

![[Blender.assets/image-20240208232210335.png]]



### 十大建模操作

#### 挤出

在编辑模式下，按 e 键挤出物体。例如一个面挤出为体

> 避免通过鼠标右键取消操作，因为这样会导致挤出的元素与原始元素重叠，为之后的建模带来麻烦。

![[Blender.assets/image-20240208233118661.png]]



#### 内插

在编辑模式下，按 i 键内插物体。例如一个面向内插出新的点、边和面

![[Blender.assets/image-20240208233340632.png]]



#### 循环切割

在编辑模式下，按 Ctrl + r 键环切，通过鼠标滚轮调整段数。环切完成后，只要没有进行其它操作，就可以通过 F9 调出参数栏。如果需要调整切割的位置，可以每次只切割一次，每次移动选择切割的位置。

> 使用 s + (x y z) 沿着这些方向对称伸缩切割。

![[Blender.assets/image-20240208233738198.png]]



#### 倒角

在编辑模式下，选中边按 Ctrl + b 键倒角，通过鼠标滚轮调整段数。倒角完成后，只要没有进行其它操作，就可以通过 F9 调出参数栏。

![[Blender.assets/image-20240208233605991.png]]



#### 合并

在编辑模式下，选择两个点，按住 m 键选择合并操作。按距离合并会合并在设定距离范围内的点。

![[Blender.assets/image-20240209145649270.png]]



#### 断开

在编辑模式下，选择一个**至少有三条边**的点，点击 v 可以断开与其它边的连接。

![[Blender.assets/image-20240209145808038.png]]



#### 填充

在编辑模式下，选择一条边，然后按住 Alt 双击循环选择一圈边，点击 f 填充面。

![[Blender.assets/image-20240209150102749.png]]

也可以选择 Ctrl + f 栅格填充，但是需要注意只能填充偶数个边组成的环。

> 对于奇数个边的情形，可以通过选中边，右键“细分”得到多一个边。

![[Blender.assets/image-20240209150500897.png]]



#### 切刀

在编辑模式下，点击 k 进入切刀模式，可以任意切割物体，按回车键退出切刀模式。

![[Blender.assets/image-20240209151302904.png]]



#### 桥接

在编辑模式下，按住 Shift 通过 Alt 键选中两个循环边，然后 Ctrl + e 选择“桥接循环边”。注意桥接只能**用于同一个物体**，因此需要 Ctrl + j 将两个物体合并为一个。

![[Blender.assets/image-20240209151611519.png]]



#### 分离

在编辑模式下，进入透视模式或线框模式，选中物体后按 p 选择“选中项”可以将其分离。

![[Blender.assets/image-20240209160544960.png]]

* 渲染：设定渲染时的细分次数

> 使用快捷键 Ctrl + 2 可以快速创建 2 次细分的细分修改器。

#### 晶格形变

首先需要创建晶格

![[Blender.assets/image-20240224145847453.png]]

用晶格框住要变形的物体（不要框住镜像物体），调整晶格的分辨率

> 注意晶格的调整变形需要在物体模式下进行。

![[Blender.assets/image-20240224151139140.png]]

然后**先选中物体，再选中晶格，Ctrl + p 选择晶格形变**，就完成了绑定。此时物体已经具有晶格修改器，调整晶格就可以对物体形变。



#### 曲线

首先创建一条路径曲线，可以在编辑模式下开启法向，显示曲线的方向。

* 应用对物体和路径曲线的所有变换后再添加曲线修改器；
* 曲线的原点最好也在坐标原点；
* 确保在 1 正视图下可以看到曲线的法向，这样才能保证 Z 轴形变在正视图内；

为物体添加曲线修改器，绑定路径曲线，设置形变轴，就可以通过调整曲线形状改变物体形状。

![[Blender.assets/image-20240224164513575.png]]



#### 蒙皮

使用蒙皮修改器为线段提供表皮。首先创建一个平面，进入编辑模式，按 m 进入合并菜单，将顶点合并到中心。

![[Blender.assets/image-20240224175256415.png]]

利用得到的顶点，挤出几条边，然后可以添加蒙皮修改器。进入透视模式，修改顶点位置就可以调整蒙皮效果。

![[Blender.assets/image-20240224175403528.png]]

选中顶点，使用 Ctrl + a 修改截面大小

![[Blender.assets/image-20240224175610879.png]]



#### 置换

![[Blender.assets/image-20240224181047932.png]]

进入编辑模式，可以开启置换修改器中“显示修改器结果”的图标，这样可以在编辑同时查看形状变化

![[Blender.assets/image-20240224182142052.png]]

可以点击置换修改器最右侧的图标，设置置换纹理图像

### 操作技巧

#### LoopTools

以 LoopTools 为例，编辑模式下选中多边形面右键细分，然后**按 n 找到编辑中的 LoopTools**，选择圆环，就可以将得到细分正多边形

![[Blender.assets/image-20240223214519599.png]]



#### 吸附投射

首先开启小磁铁图标，选择“面投射”和“旋转对齐目标”

#### 衰减模式

使用 o 或者磁铁吸附图标右边的圆形图标开启衰减模式，此时进行平移等变换会衰减应用到周围的元素

#### 快速拉平

当我们需要将底部的边都拉平时，可以选中这些边，然后通过 s + z + 0 将它们拉平。这实际是**将选中边沿着 z 方向伸缩 0 倍**，因此相当于将它们全部拍平。

> 注意确认没有开启衰减模式。

![[Blender.assets/image-20240224013300002.png]]



#### 旋绕工具

在编辑模式下可以开启旋绕工具。选中一个面，设定**游标位置**。旋绕工具会将选中元素绕着游标位置转动，得到连续的物体

![[Blender.assets/image-20240224014836739.png]]

如果勾选“使用副本”，就会得到离散生成的元素。可以调整旋转中心的位置

### 贝塞尔曲线

贝塞尔曲线使用控制杆修改形状。创建一条贝塞尔曲线，然后进入编辑模式，可以选择点，用 r, s, g 对控制杆进行旋转、缩放和平移。使用 e 挤出顶点来创建新的顶点。

> 选择曲线上一点，Alt + c 即可闭合曲线。使用 x 键可以删除顶点（保持连续），删除段数（断开曲线）。

NURBS 曲线使用控制点修改形状。创建 NURBS 曲线，进入编辑模式，选择控制点后，按 n 可以看到 W 即权重。

最常用的是背景节点和环境纹理节点。

### 场景布光

#### 定基调

根据实际场景选择环境光，例如室内环境使用对应的 HDR 贴图。借助映射节点，将贴图旋转，让物体位于窗户附近。布置完成后，勾选“胶片-透明”选项，取消背景渲染

![[Blender.assets/image-20240229201934451.png]]



#### 三点布光

使用主光、辅助光、轮廓光进行三点布光。

* 主光负责照亮主体
* 辅助光负责照亮暗部
* 轮廓光一般从背后照射，负责凸显轮廓

主光一般亮度最高，辅助光和轮廓光较弱，主要照亮阴影。为了防止轮廓光产生的逆光投出阴影，可以将其“阴影”选项取消勾选。

![[Blender.assets/image-20240229203047394.png]]



### 阴影效果

要实现灯光透过指定纹理投影的效果，可以利用“透明 BSDF”节点。指定纹理为噪波纹理，将它作为系数，形成蒙版混合效果；分别为原理 BSDF 和透明 BSDF 指定颜色，然后混合，这样就会根据噪波纹理进行透射：黑色纹理部分不会透射，而白色纹理部分会透光。

> 需要在 Cycles 渲染下才能预览渲染效果。

![[Blender.assets/image-20240229211659122.png]]



### 导入素材

Blender 有两种导入 blender 素材的方式：

* 关联：导入素材，但是不能修改。好处是原 blender 文件中的素材修改时，当前关联的素材也会修改（需要“文件-重新加载”，记得先保存）
* 追加：导入素材，可以修改

导入方式是双击 .blend 文件，选择内部的物件或集合中的物件，然后导入。

如果需要导入其它格式的文件，则使用“导入”选项。

### 脚本编辑器

切换到脚本窗口，可以建立脚本。但是脚本窗口最重要的作用是提供左下角的属性变量：当修改右侧的属性时，对应的变化会出现在左下角。这便于我们通过 py 脚本来控制属性。

### 自动化渲染

通过 python 配合 powershell 来实现自动化批量渲染。首先需要安装 bpy 库

```shell
$ pip install fake-bpy-module
```

然后可以设置渲染参数

```python
import bpy

scene = bpy.data.scenes["Scene"]
scene.render.engine = 'CYCLES'
scene.render.image_settings.color_depth = '8'
scene.render.image_settings.color_mode = 'RGB'
scene.render.image_settings.file_format = 'OPEN_EXR'
scene.render.image_settings.exr_codec = 'NONE'
scene.render.resolution_x = 512
scene.render.resolution_y = 512
scene.render.filepath = 'E:\\Desktop\\'
scene.cycles.samples = 1024
```

可以在偏好设置中开启 Python 提示，这样鼠标悬停在属性上就会显示对应的属性变量。

![[Blender.assets/image-20240312190128832.png]]



### 生成节点

首先需要启用节点

```python
bpy.data.scenes['Scene'].use_nodes = True
```



可以通过脚本来获得节点和控制节点。不同的着色器中的节点通过不同的属性获得，例如

```python
tree = bpy.data.scenes['Scene'].node_tree
```

这对应着合成器中的节点树。再例如

```python
world = bpy.data.worlds['World'].node_tree
```

这对应着世界环境中的节点树。如果不清楚节点树对应什么属性，可以创建一个节点，查看对应属性的获得方式，就可以得到节点树的获得方式。

![[Blender.assets/image-20240409230429926.png]]



获得节点树后，可以增加节点

```python
image_node = tree.nodes.new('CompositorNodeImage')			# 创建图像节点
image_node.image = bpy.data.images.new('src', 512, 512)		# 生成图像，指定名称和大小

image_node.image.source = 'FILE'							# 指定图像的来源，FILE 表示图像来自一个文件
image_node.image.filepath = 'E:/Desktop/girl.jpg'			# 指定图像文件的路径
image_node.image.reload()									# 重新从来源载入图片

image_node.label = 'InputImage'								# 设置节点标签，也就是
```

切换并创建一个文本后，可以执行脚本。例如上述操作会得到下面的结果

![[Blender.assets/image-20240409231059811.png]]



对于可以查看属性的节点，只需要移动上去，就可以看到对应的节点名。例如下面的节点为 `CompositorNodeComposite`，通过一个勾选属性可以看到。

![[Blender.assets/image-20240409232142232.png]]

而对于没有可操作的属性的节点，就需要查阅手册。



### 连接节点

节点之间需要将属性对应连接，这需要通过 links 属性。例如再创建两个节点，然后将一个节点的输出链接到另一个节点的输入。

* 输出属性为 outputs，是一个字典，可以通过下标或者键来获取值（节点）
* 输入属性为 inputs，是一个字典，可以通过下标或者键来获取值（节点）

下标的顺序与属性的顺序一般一致。例如

```python
links = tree.links

# 降噪节点
denoise_node = tree.nodes.new(type="CompositorNodeDenoise")
denoise_node.label = 'Denoise'
links.new(image_node.outputs[0], denoise_node.inputs[0])

# 合成器节点
composite_node = tree.nodes.new(type="CompositorNodeComposite")
composite_node.label = 'Composite'
links.new(denoise_node.outputs[0], composite_node.inputs[0])
```

可以通过脚本直接渲染图像。按照顺序调用

```python
bpy.ops.render.render()
bpy.data.images["Render Result"].save_render(path)
```

其中 images 是一个字典，可以在渲染结果窗口处看到所有可用的键。这里 `"Render Result"` 是默认输出对应的键，可以获得渲染结果。

#### 空物体

使用空物体来确定阵列修改器的移动效果。首先创建一个空物体轴，然后给要操作的物体添加阵列修改器，设为“物体偏移”，吸取空物体轴，这样就可以通过操作空物体来改变移动效果：**针对空物体的移动、伸缩、旋转操作会叠加到每个阵列生成物体上**。

> 由于会叠加变换，注意确保物体本身的所有变换都已经应用。

![[Blender.assets/image-20240224152445127.png]]



使用空物体-立方体可以作为复杂模型的父组：框选复杂模型，再选择空物体，Ctrl + p 设为父组。这样就可以通过空物体方便地移动复杂模型，尤其是带有阵列修改器的模型。

![[Blender.assets/image-20240224154539390.png]]



#### 关联复制

使用 Alt + d 复制物体，则对原始物体的所有更改都会影响所有复制体。



#### 曲线吸附

通过关联复制出一侧的叶片，再 Ctrl + d 复制出另一侧的叶片。选中所有叶片，最后选中杆，Ctrl + p 选择“顶点（基于三点）”。最后为杆添加曲线修改器，则杆会沿着曲线形变，叶片会吸附在杆上形变。

> 注意调整路径曲线时，将原点移动到坐标原点。不要调整过大，防止出现变形。如果调整过程中，叶片整体突然转动到另一个平面，可以切换到该平面调整，使其恢复到正视图。（在正视图左右扭动时叶片不会脱离杆，而前后扭动可能导致分离）

一定要**确保叶片自身的旋转、缩放变换都已经应用！！！**

![[Blender.assets/image-20240224174333495.png]]

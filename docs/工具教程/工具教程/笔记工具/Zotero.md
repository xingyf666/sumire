# Zotero

## 基本介绍

Zotero 是一个文献管理工具，通过装载插件实现丰富的文献管理操作。不同版本之间的插件并不兼容，我们使用最新的 `Zotero7` 版本。



### 插件管理

从 `7.0` 版本开始，可以使用 `addons` 插件管理市场来安装更新插件。在 github 搜索并下载插件，通过“工具-插件”安装。

![](image-20241003234116616.png)

安装完成后，在“工具”中找到“插件市场”打开，寻找需要的插件进行安装即可。

# NSIS

需要使用管理员权限打开编辑器 `NSISEditer.exe`（之前尝试用这个生成，似乎有问题，还是使用NSIS启动器中的第二个编辑器生成脚本，用这个编译器编译比较好）。



点击“新建向导”，自动创建可视化脚本，修改名称、作者，网站可以清除

![](image-20210909195340222.png)



修改安装图标

![](image-20210909195636161.png)



清空授权框，不使用授权文件

![](image-20210909195721901.png)



点击加号添加源程序，点击双加号添加资源文件，选中文件夹，勾选“包含子目录”和“单独添加每个文件”

![](image-20210909195909306.png)

之后按照流程进行即可，可以自定义卸载的图标；需要注意，编译运行脚本之前，不能随便改脚本的位置，否则脚本中定义的路径宏可能会出错，导致编译不通过（这个编辑器编译开头是乱码，导致如果出错，看不出来是什么错误）。

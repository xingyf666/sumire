# 过渡方法

## 基本介绍

有必要区分不同的过渡方法 blending

- 粗糙混合 superficial blending，没有显式数学公式。这类混合对应圆角方法
- 曲面混合 surface blending，光滑连接给定曲面
- 多面体混合 polyhedral blending
- 体混合 volumetric blending，实体系统混合

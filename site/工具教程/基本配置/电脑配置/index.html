    电脑配置

    修改菜单样式

    在命令行输入

    reg add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve

    修改菜单为 win10 样式，然后在任务管理器重启资源管理器即可。

    如果想恢复 win11 样式，输入

    reg delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /va /f

    然后在任务管理器重启资源管理器即可。

    安装字体

    将字体文件移动到 C:\Users\xyf\AppData\Local\Microsoft\Windows\Fonts 目录下，右键字体选择“为所有用户安装”，然后就可以使用该字体。

    分辨率

    在高分辨率屏幕上运行低分辨率程序时，可能会有显示过小的问题。右键属性-兼容性-更改高 DPI 设置-勾选高 DPI 缩放替代-系统增强。

    查看占用

    删除文件夹时，有时会出现文件夹被占用的情况，可以打开任务管理器，切换到“性能”选项卡，打开资源监视器

    ![[image-20240706212412080.png]]

    在资源监视器的“关联的句柄”中输入文件夹路径，即可查看所有占用此文件夹的进程。

    ![[image-20240706212438799.png]]

    电源管理

    查看电池

    打开 cmd 输入

    powercfg /batteryreport

    生成电池使用报告。

    设置模式

    显示处理器性能提升模式选项

    \HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Power\PowerSettings\54533251-82be-4824-96c1-47b60b740d00\be337238-0d82-4146-a960-4f3749d470c7

    将 Attributes 从 1 改为 2 即可。

    显示最大处理器状态选项，命令行执行

    reg add HKLM\System\CurrentControlSet\Control\Power /v PlatformAoAcOverride /t REG_DWORD /d 0

    网络适配器

    防止休眠

    为了防止网络在休眠时断连，首先以管理员权限打开 PowerShell，执行

    reg add HKLM\System\CurrentControlSet\Control\Power /v PlatformAoAcOverride /t REG_DWORD /d 0

    现在进入网络适配器设置，取消 WIFI 设备的休眠设置

    ![[image-20241018185038970.png]]

    VPN 连接

    有可能出现 VPN 创建后无法连接的问题

    解决L2TP连接尝试失败，因为安全层在初始化与远程计算机的协商时遇到一个处理错误

    这时候打开注册表，找到目录

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\PolicyAgent

    创建一个 DWORD 值 AssumeUDPEncapsulationContextOnSendRule 设为 2；然后找到

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\RasMan\Parameters

    创建一个 DWORD 值 ProhibitIPSec 设为 1；然后创建/修改 AllowL2TPWeakCrypto 值为 1 。

    进入计算机管理，找到 IPsec Policy Agent 和 Routing and Remote Access，右键属性改为自动

    ![[image-20241018190526770.png]]

    最后进入控制面板，找到网络连接

    ![[image-20241018191535746.png]]

    右键 VPN 属性，找到“安全”，按照下面勾选

    ![[image-20241018191609707.png]]

    重启计算机尝试连接即可。

    无网络问题

    打开设备管理器，找到网络适配器，注意到下面这个设备

    Qualcomm Atheros QCA61x4A Wireless Network Adapter

    图标上有一个黄色三角图标，只需要右键卸载设备，然后右键网络适配器，扫描硬件改动就可以重新安装 建议右键设备属性，将电源管理中的允许关闭电源的选项取消勾选。

    磁盘合并

    虚拟内存

    有时格式化磁盘时，会显示磁盘被占用，需要进入高级系统设置

    ![[image-20260318194623914.png]]

    选择“高级”选项卡，点击第一个“设置”

    ![[image-20260318194653239.png]]

    选择“高级”选项卡，点击“更改”

    ![[image-20260318194723896.png]]

    取消勾选“自动管理”，然后选择被占用的磁盘，将其改为“无分页文件“，点击“设置”

    ![[image-20260318194803041.png]]

    完成后，重启电脑即可格式化。

    [!note] 注意之后还需要重新分配虚拟内存，防止电脑卡顿，部分应用甚至会崩溃。

    合并分区

    使用 DiskGenius 合并空闲分区，直接右键选择空闲分区，分配给 C 盘即可

    [!note] 需要先解锁 BitLocker 才能合并不相邻的分区。

    ![[image-20260319000558073.png]]

    打开方式

    打开方式中可能存在多余的已卸载程序，在注册表的这一路径下删除对应的程序信息即可

    计算机\HKEY_CLASSES_ROOT\Applications\

    解除占用

    如果文件/文件夹被占用，可以打开“资源监视器”，在“CPU”选项卡“关联的句柄”中输入对应的路径，等待片刻就可以看到占用程序

    ![[image-20260321224603557.png]]

    出于奇怪的原因，电脑可能出现睡眠后无法唤醒的问题，可以先下载修复工具

    解决未激活

    可能电脑突然显示系统未激活，也许是因为使用第三方激活工具的问题。可以在 PowerShell 输入代码

    irm https://massgrave.dev/get | iex

    执行命令后输入 1 就可以自动激活。

    Net Framework

    对于比较古老的 win10 系统版本，可能出现 net framework 4.8 无法安装导致 vs 也无法安装的问题，以及不能用 . 开头命名文件的问题，都是因为系统没有更新到最新版本，进行 windows 更新即可解决。

    Rhino

    基本介绍

    标准工具

    Rhino 的窗口分为 4 个部分，分别展示不同视图。鼠标左键可以框选，鼠标右键可以拖动视角。

    ![[image-20230828104042306.png]]

    按下 Shift 键，通过鼠标右键可以平移视图；按下 Ctrl 键，通过鼠标右键可以缩放视图；按下 z + space 键，框选可以放大指定位置。

    在标准工具栏中可以找到各种进行视图窗口操作的方法，例如重置为默认 4 工作视窗

    ![[image-20230829220635033.png]]

    还可以调整着色方法为线框模式或着色模式等

    ![[image-20230829220727681.png]]

    操作轴

    在最下方的工具栏启用操作轴可以让控制点移动更加便捷

    • 通过箭头可以调整位置
    • 通过箭头上的圆点可以挤出图案，例如移动点形成直线
    • 通过弧线可以旋转物体

    ![[image-20230829224143588.png]]

    点击弧线操作轴后，可以输入角度值进行旋转

    ![[image-20240329152844979.png]]

    智慧轨迹

    在最下方的工具栏启用智慧轨迹在绘制曲线或直线时，可以进行辅助。例如绘制多段直线时，移动到顶点位置后，可以沿着垂直轨迹线移动点的位置

    ![[image-20230829224708765.png]]

    不仅如此，还可以按下 Ctrl 键来进行定位。例如绘制直线时，边移动顶点边点击 Ctrl 键会留下白色轨迹点，可以沿着白色轨迹点产生的垂直轨迹线移动点

    ![[image-20230829231100806.png]]

    物件锁点

    同样的，在最下方工具栏启用物件锁点，就会显示如下的选项。选中端点、点，就可以对点进行捕捉

    ![[image-20230829231704488.png]]

    例如绘制一条直线，然后以这条直线的端点作为起点绘制另一条直线

    ![[image-20230829231848806.png]]

    建构历史

    建构历史会记录一个物件被建立的过程，之后产生的形状会受到之前的影响。例如开启建构历史后，用直线挤出面

    ![[image-20230829235003104.png]]

    然后修改原始直线进行重建，那么整个被挤出的形状也会随着原始直线发生变化

    ![[image-20230829235102295.png]]

    工具列

    在标准工具界面下，左侧上方有 3 个工具列，分别是点工具列、直线工具列和曲线工具列。

    ![[image-20230829225426165.png]]

    直线工具列

    正常情况下，所有绘制直线、曲线都会在工作平面内，如果需要绘制具有高度的点，可以按住 Ctrl 键后再点击，就可以沿着垂直方向移动

    ![[image-20230829231231644.png]]

    使用 Shift 可以沿着坐标轴移动

    ![[image-20230829232515729.png]]

    可以使用 Tab 键捕捉当前方向，例如绘制多段直线，希望沿着之前的方向绘制。那么可以先反过来捕捉之前的端点，然后点击 Tab 键捕捉方向，就可以沿着之前的线段方向绘制

    ![[image-20230829232013872.png]]

    指令可以进行点击操作，例如绘制多重直线时，指令栏将会显示当前命令。括号中的内容是对应的参数选项，可以点击进行开启或关闭，甚至进行设置。这里如果开启导线，就可以实现沿着之前的方向绘制的效果

    ![[image-20230829232239315.png]]

    曲线工具列

    最常用的就是控制点曲线，可以产生 3 次样条曲线

    ![[image-20230829233816275.png]]

    非常重要的功能是曲线重建，可以对任何直线、曲线进行阶数、节点数修改

    ![[image-20230829233944160.png]]

    曲面工具列

    创建基本曲面使用“指定三或四个角建立曲面”，然后进行曲面重建，通过 F10 显示控制点进行调整

    ![[image-20230830002329852.png]]

    还可以通过平面曲线创建面。首先绘制多段封闭直线，然后点击“以平面曲线建立曲面”，就得到修剪平面曲面。显示控制点后可以看到整张曲面以及修剪线

    ![[image-20230830002843677.png]]

    点击曲面，通过鼠标右键取消修剪

    ![[image-20230830003123208.png]]

    可以进行修剪，例如生成平面曲面，建立裁剪曲线

    ![[image-20230830003330469.png]]

    鼠标左键点击修剪，选择裁剪曲线后按 Enter，然后点击曲面要裁剪的部分后按 Enter 即可修剪

    ![[image-20230830003839572.png]]

    在曲面工具栏中可以找到节点插入方法以及显示、调整控制点的方法

    ![[image-20230830004736390.png]]

    执行命令

    在最上方的指令输入框中可以输入指令，然后点击空格即可执行指令。

    命令 PointsOn/PointsOff 可以显示和隐藏控制点，也可以通过快捷键 F10/F11 实现相同的效果。

    文件读写

    导入 obj

    导入 obj 时最好设置为如下选项，不使用材质颜色，防止材质丢失导致不能正常显示。

    ![[image-20231210171231696.png]]

    导出 obj

    导出 obj 时应该根据需要设置，例如在导出 NURBS 物件时应该设置，为 NURBS 格式，并且不导出材质。

    ![[image-20231210171402385.png]] ![[image-20231210171433183.png]]

    命令操作

    创建线段

    使用 line 命令创建单个线段，分别输入 0 作为起点，10 作为长度

    ![[image-20240329144334804.png]]

    按住 Shift 键可以让角度按照 $90^\circ$ 的整数倍变化，得到沿着坐标轴的直线

    ![[image-20240329144538751.png]]

    分割线段

    可以对完整的曲线进行分割

    ![[image-20230829234156056.png]]

    首先绘制一个封闭方形，点击这个功能，可以看到命令要求选择物件，选中方形然后按 Enter 完成

    ![[image-20230829234259994.png]]

    这时候会要求选择切割用的物件，由于我们切割曲线，因此选择点（输入 P）。然后可以选择曲线上的分割点，按 Enter 即可分割完成

    ![[image-20230829234346938.png]]

    也可以使用抽离子线段功能分离多重线段的每个部分。

    环形阵列

    通过“矩阵阵列”命令中的环形阵列

    ![[image-20240329144630990.png]]

    或者 ArrayPolar 命令创建阵列

    ![[image-20240329144739720.png]]

    依次指定阵列物体、中心点、阵列数量 3、总的旋转角 360，就可以得到一个 Y 形的阵列

    ![[image-20240329144913129.png]]

    圆管操作

    使用圆管操作 pipe 从曲线创建圆管实体对象

    ![[image-20240329145424125.png]]

    炸开曲面

    炸开操作用于将实体的表面分割为多个曲面，例如将圆管炸开为 3 张曲面

    ![[image-20240329145749945.png]]

    抽离结构线

    使用“抽离结构线”将曲面拆分成结构线的线框（使用它左侧的命令可以只抽离一条结构线）

    ![[image-20240329163744043.png]]

    结构线分割

    当曲面具有结构线时，可以通过结构线拆分曲面。依次设定切割用物件为“结构线”、分割方向为 U 方向，回车进行分割。

    分割时可以利用原本辅助线的“中点”锁定，确保从中间分割。

    ![[image-20240329150224444.png]]

    开启“交点锁定”，可以方便地根据结构线网格切分曲面

    ![[image-20240329152203382.png]]

    分割/合并边缘

    开启“四分点”锁定，使用分割边缘来切分曲面边界

    ![[image-20240329151116043.png]]

    注意右键点击可以将边缘重新合并

    ![[image-20240329154634665.png]]

    混接曲线

    使用“可调式混接曲线”控制曲线的 G0, G1, G2 连续性衔接

    ![[image-20240329154337251.png]]

    混接曲面

    使用混接曲面 blendsrf 选中分割后的边缘，生成过渡曲面

    ![[image-20240329151821622.png]]

    曲线延伸

    选中端点，使用“曲线延伸”来延长曲线甚至结构线

    ![[image-20240329152402605.png]]

    物件求交

    通过求交来得到需要的结构线信息

    ![[image-20240329153841144.png]]

    曲线挤出

    使用曲面工具中的“挤出”操作来确定挤出方向和挤出长度。虽然使用操作轴也可以挤出，但是无法沿着预期的方向挤出。

    ![[image-20240329162226127.png]]

    XNurbs

    将 XNurbs 插件拖放到 Rhino 窗口中，可以在右上角唤出插件栏，将其拖放固定到工具栏

    • XNurbsRhino 将会停放在新一行工具栏上
    • XNurbsRhino 5.0 XNurbsRhino 将会创建新的工具栏

    ![[image-20240812010655095.png]]

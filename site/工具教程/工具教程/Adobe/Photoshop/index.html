    Photoshop

    基本介绍

    选项设置

    标尺

    为了方便像素操作，我们在“编辑-首选项-单位与标尺”中修改标尺单位为“像素”。

    参考线

    在“视图-显示”中修改显示网格和智能参考线。

    分辨率

    • 新建不打印图像的分辨率最好给 72，采用 RGB 模式（Red Green Blue）
    • 新建打印图像的分辨率最好给 300，采用 CMYK 模式（青色 品红 黄色 黑色）

    快捷键

    需要注意设置输入法为英文，否则一些快捷键会无效。

    单击键

    单击键 作用 单击键 作用
    x 切换前景色和背景色 d 恢复初始前景色和背景色
    F5 画笔预设 e 橡皮擦
    g 油漆桶/渐变 c 裁剪
    m 选框 t 文字
    v 移动（默认） Esc 撤销自由变换

    组合键

    组合键 作用 组合键 作用
    Ctrl + 0 让图像适合屏幕 Ctrl + 1 按像素比例 1:1 显示
    Ctrl + j 新建指定图层的拷贝图层 Ctrl + n 新建文件
    Ctrl + [ 选定图层向下移动 Ctrl + ] 选定图层向上移动
    Ctrl + e 合并图层 Ctrl + r 调出标尺，辅助线功能
    Ctrl + d 取消选择 Ctrl + Delete 填充背景色
    Ctrl + t 自由变换
    Ctrl + Shift + f 渐隐修补选区 Ctrl + Shift + x 液化
    Ctrl + Shift + u 去色 Ctrl + Shift + d 重新选择
    Ctrl + Shift + e 合并可见图层 Ctrl + Shift + l 反选
    Shift + F5 选择对图层填色 Shift + Ctrl + ; 对齐
    Shift + Ctrl + z 前进一步
    Alt + Ctrl + z 后退一步 Alt + Delete 填充前景色
    Alt + Ctrl + g 新建剪切蒙版 Alt + Ctrl + Shift + e 创建所有可见图层合并后的新图层

    长按键

    按键 作用 按键 作用
    Alt 滚轮缩放图像 Space 拖动图像
    Shift 平整拖动 Ctrl 左右滚动图像

    图层

    图层名称从上到下排列，图层位置也从上到下排列。如果选中了某些菜单项，例如选择“类型”项，注意到出现了一个蓝色边框，这时候使用快捷键进行操作时可能出现问题，需要先按回车完成操作。

    对图层操作时，需要先解锁图层

    • 按住 Shift 键再拖拽其他图片，可以将其他图片放在指定图片的中央；
    • 按住 Alt 键再拖拽图层，会复制图层；
    • 按住 Ctrl 键再点击图层，可以选定这一图层进行操作；
    • 按住 Ctrl 键再点击组，可以选定这一组进行操作；

    不可见图层不会受到操作影响，锁定图层不能打组。复制图层会复制整个图层，复制图层可以复制所选图像为一个单独的图层。

    操作技巧：

    1. 按住 Alt 键，点击图层前的眼睛即可只查看当前图层；再按住 Alt 键，点击即可恢复查看。
    2. 调节两张图的位置关系时，可以降低上方图片的不透明度。

    文字

    使用文字工具，点击图中位置就会创建一个文本框。如果需要创建一个很大的文本框，可以点击以后拖拽

    文字图层不能与其它图层合并，需要先右键选择栅格化文字，将文本变为图像以后再合并。

    进阶教程

    选区

    选区时，按住 Shift 键选取正方形/圆，按住 Alt 键以鼠标位置为中心选取（单行单列选取是选取一行/列像素，一般用不到）。选区后

    • 按住 Shift 键进行加选；
    • 按住 Alt 键进行减选；

    如果希望选取边缘有渐变，先设置选取菜单中的羽化值，然后选区；或者在选区后通过右键设置羽化。

    • 关闭选区后，可以用“选择-重新选择”来复原选区

    操作技巧：可以选取一个图层，然后对另一个图层进行裁剪。

    使用直线工具绘制图形后，可以右击建立选区。

    操作技巧：可以 Ctrl + A 选择整个图层。

    自由变换

    Ctrl + t 进入自由变换时，按住 Alt 键再拖拽图层是以中心点为原点进行缩放，按住 Shift 键是对一条边缩放，同时按住 Shift 键和 Alt 键是以中心点为原点等比例缩放。

    常用操作：按住 Ctrl 键直接切换到扭曲。

    调小图片，再调大图片就会失真。尽量不要把图片调大调小，避免失真。

    套索工具

    最常用的套索工具是磁性套索，选择起点后鼠标移动套索即可。

    按 Esc 键取消套索，Backspace 键退回到上一个点，空格键切换到抓手工具移动图像，按 Alt 键进行套索（套索过程中可以松开 Alt 键）来减选，按 Shift 键进行套索（套索过程中可以松开 Shift 键）来加选。

    设置项：

    • 频率：改变点的间隔
    • 对比度：改变选取时对颜色差异的要求
    • 宽度：套索的有效范围

    使用磁性套索获得选区，然后使用多边形套索进行局部修正。完成选取后，点击套索菜单中的“选择并遮住”进行边缘调整模式

    通常设置为

    • 平滑：10 以内
    • 羽化：看情况
    • 对比度：10 以内
    • 移动边缘：-5 左右（让边缘收缩）
    • 净化颜色：80 以上（去除多余颜色）
    • 输出到新建带有图层蒙版的图层

    这时候会得到一个复制图层和选取蒙版，右键点击蒙版，选择应用图层蒙版，才能得到裁剪后的图层。

    快速选择工具

    用于自动选取，点击指定的位置就能快速选择对比度类似的区域。默认为加选状态。

    • 当加选时，按住 Alt 键暂时切换减选；
    • 当减选时，按住 Shift 键暂时切换为加选

    按键 [ 减小画笔，按键 ] 增大画笔。

    魔棒工具

    容差值越大，所选取的颜色范围越大，反之越小。因此魔棒工具常用于背景较单一的图片，利用背景反选内容。

    色彩范围工具

    点击“选择-色彩范围”选项在图片上选取颜色，自动选取与颜色相近的部分（在图中标示为白色）。

    修图

    裁剪切片工具

    调整操作基本与自由变换一致，按 Esc 键取消裁剪。当图像过大时，可以使用切片工具，右击图片-划分切片后，点击“文件-导出-储存为 Web 所用格式”，然后选择 JPG 格式储存。

    图像-图像大小：改变所选图像的大小

    图像-画布大小：改变所选图像所在画纸的大小

    裁剪工具也可以放大画布。

    内容识别缩放

    选择“编辑-内容识别缩放”，在面板最右侧选中保护肤色，这样在自由变换时可以保护人物不变形

    但这种方式能起到的保护效果仅限于可识别的人体，更好的方案是使用选区保护。

    在图层右边的通道中创建新通道，并将保护区域涂成白色，再使其不可见，选择其余通道，返回到图层界面。这时再选择内容识别缩放，面板最右侧保护，选择相应通道即可

    内容识别填充

    选择图层后，使用选区工具选择水印，再使用编辑-填充-内容识别 Shift + F5 去除水印。

    操控变形工具

    用于对人物动作进行微调，选择“编辑-操纵变形”，钉子选定不可移动的位置，长按移动变形

    画笔

    按键 [ 调小画笔，按键 ] 调大画笔。

    操作技巧：先点击一个点，按住 Shift 键，再点击另一个点可以画一条直线。

    利用这一技巧，我们可以实现邮票边缘的绘制。首先通过裁剪工具扩大画布范围，创建两层背景。

    然后调整画笔间距，绘制锯齿边界

    历史记录画笔

    历史记录画笔可以绘制之前记录的背景，而历史记录艺术画笔可以增加艺术效果。例如填充一个白色背景，然后使用历史记录艺术画笔，将画笔调小，然后就可以绘制油画效果。

    橡皮擦工具

    橡皮擦是一种绘画背景色的画笔，如果背景图层未锁定，则直接擦除为透明图层！橡皮擦功能和画笔基本一致，除了魔术橡皮擦工具，它实际上是等价于魔棒工具 + Delete 操作。

    混合模式

    图层混合模式只对活动图层有效，需要解锁图层。它会混合当前图层下面的所有图层，因此要将素材放在上面，被混合的图层放在下面。混合以后，可以用橡皮擦擦除其中不希望混合的人物等图像。

    图层样式只对活动图层有效。右键图层选择“混合选项”打开图层样式面板，取消勾选样式即可取消对应的样式。图层样式可以拷贝粘贴。

    投影

    图层投影，制造虚化效果，“扩展”控制投影大小，“大小”控制虚化程度。

    投影的另一种常见用法是制作倒影：

    1. 首先将物体复制翻转，通过自由变换调整大小，然后填充成背景色
    2. 给这个复制体设置投影，就得到影子效果
    3. 再次将物体复制翻转，设置不透明度，通过低流量的橡皮擦擦除得到倒影效果

    描边

    填充不透明度

    在混合选项中将“填充不透明度”设为 0，此时文字内部完全透明。注意不能设置“不透明度”，否则整个文字包括样式会全部透明。然后可以利用这一点设置投影样式，从而实现不易去除且不影响观看的水印效果。

    钢笔

    路径模式

    选择钢笔工具，调整为“路径”模式，然后可以创建一个路径，路径是虚拟的线条，不是真实的。

    钢笔路径可以提供曲线选区

    • 当钢笔位于路径上时，点击添加锚点；
    • 当钢笔位于锚点上时，点击可以删除锚点；
    • 按下 Esc 键取消选择路径，再按下 Esc 键取消路径；
    • 按住 Alt 键，移动到锚点，会变为转换点工具，此时可以拖动操作柄；
    • 按住 Ctrl 键，先对锚点进行框选，不松开 Ctrl 键，可以移动锚点；

    可以右击/在面板上选择建立选区。

    形状模式

    形状模式用于绘制图形（内部填充），生成图层。面板调节填充图案、颜色、渐变，描边图案、颜色、渐变，以及描边线段的样式。最重要的是合并、减去、相交等形状操作。

    例如使用相交操作，先绘制一个圆角矩形，然后设置相交选项，最后用钢笔形状路径绘制四边形得到相交部分。

    矩形工具

    矩形工具是不同于钢笔工具的工具，也具有形状模式和路径模式，其效果与钢笔差不多。矩形工具能够绘制规则的路径，而钢笔工具则绘制锚点控制的路径。

    文字路径

    在路径上使用文字工具，文字会跟着路径走。例如绘制一个圆角矩形路径，选择文字工具移动到路径上，点击即可创建沿着路径的文字。

    蒙版

    蒙版是用于调节图像修改程度以及进行局部调整的命令。蒙版只有三种颜色：

    • 黑色代表保护；
    • 白色代表允许修改；
    • 灰色的程度用来调整被修改的程度；

    点击面板右下角“fx”右边的图标创建蒙版（默认为白色）；按住 Alt 键，点击新建蒙版，可以创建黑色蒙版。

    • 按住 Alt 键点击蒙版缩略图，可以查看当前蒙版状态；
    • 再按住 Alt 键点击，即可恢复图层状态；

    按住 Shift 键点击蒙版可以停用蒙版；重复操作可恢复开启状态。图层和蒙版之间上锁时任何缩放操作同时有效，解锁后互不影响。

    使用技巧是：创建白色蒙版，此时白色背景覆盖了后面的图层。然后使用黑色画笔（前景色为黑色）选择蒙版进行绘制，则变成黑色的部分就会被“保护”，即对应的白色背景被隐藏，露出后面的图层。

    更进一步，如果修改画笔的不透明度，然后进行绘制，就可以调节保护程度。每绘制一层，后面图层的颜色就会加深一层，便于微调。

    反之，也可以填充为黑色背景，使用白色画笔进行绘制。

    蒙版换脸

    使用蒙版可以进行换脸操作：首先将目标人物的头像放在上一层，创建黑色蒙版。然后使用半透明的白色逐渐解除保护，在这一过程中调整人脸的位置和大小。为了方便自由变换，在对齐人脸后可以围绕人脸选区，然后反选删除多余的图像。

    要保持两张脸的风格一致，需要在“图像”菜单中调整亮度、对比度、饱和度、色彩平衡等设置。

    调色

    使用“图像-调整”有两个缺点：

    1. 调整不可逆；
    2. 不能控制取消某个调色步骤；

    正确的调色方式是通过创建调色图层来进行的。新建的调色图层可以可逆地调整，并且它同时建立了一个白色蒙版，可以在蒙版上绘制来调整调色的范围。双击图层缩略图可以查看具体的属性。

    默认情况下，调色图层会应用到后面的所有图层，可以点击属性栏下面第一个图标设置“只影响下一个图层”。

    蒙版设置

    选择蒙版图层以后，可以调整蒙版属性。

    • “浓度”调节不透明度；
    • “羽化”模糊黑白交界的边缘；

    如果想要将某个图层的蒙版应用到另一个图层，可以先复制该图层，将蒙版拖拽到目标图层，再删除复制的图层。

    调色过程中常常需要选取调色范围，例如在蒙版中设定可修改的白色选取，这时调色就只会应用到白色区域。

    调色简介

    需要区分“亮度”和“明度”的概念。

    • 亮度：图层的亮度
    • 明度：大于零时，等价于在图层上覆盖白色图层，调节不透明度；小于零时，等价于在图层上覆盖黑色图层，调节不透明度

    几种常用的调色命令如下：

    调色命令 作用
    自然饱和度 用于调节风景的饱合度
    色彩平衡 调节颜色成分（注意勾选“保留明度”）。中间调：调整灰色区域；阴影：调整暗色区域；高光：调整亮色区域
    照片滤镜 滤镜：选择滤镜模式；颜色：得到图层透过设定颜色“玻璃”产生效果；浓度：调节颜色的浓度
    黑白 用于制作黑白照片

    色阶与曲线

    色阶面板上，黑色、灰色、白色的箭头分别表示暗部、灰部、亮部区域。

    • 黑色箭头左边的区域是最暗的区域，因此将其右移会使暗的区域更暗；
    • 白色箭头右边的区域是最亮的区域，因此将其左移会使亮的区域更亮；
    • 灰色箭头调整整体的亮度，将其左移，则亮部区域增多；右移，则暗部区域增多；

    最下方调节明度的横条一般不使用。

    曲线面板上，图表横轴是色阶，纵轴是亮度。

    • 向上调节曲线使得调节部位变亮；
    • 向下调节曲线则变暗；

    用于针对不同亮度区域的调整亮度。例如将暗部向上调亮，同时将亮部向下调整回原始亮度

    人像修图

    仿制图章工具

    选择“仿制图章工具”，按住 Alt 键，点击需要复制的位置，松开 Alt 键，然后进行绘制，就可以将复制位置的图像绘制到新的位置。可以调节画笔不透明度来制作幽灵

    修复画笔工具

    与仿制印章工具使用方法相同，但能使绘制部位的图像与原图像的颜色更加融合。

    修补工具

    绘制选区，拖拽选区。当处于源模式时，拖拽位置的图像覆盖选区位置的图像；当处于目标模式时，选区位置图像覆盖拖拽位置的图像。

    内容感知移动工具

    绘制选区，拖拽选区。当处于移动模式时，剪切选区到移动位置，原区域用周围图像自动补全；当处于扩展模式时，复制选区到移动位置。该工具使用后可以对所选图像进行自由变换。

    污点修复画笔工具

    与内容识别填充去水印功能相同，绘制区域由周围图像补全，区别仅在于可以随意涂抹补全。

    红眼工具

    拖拽选择图像，自动去除眼睛的红色（用于摄影后去红眼）。

    使模糊的图变清晰

    利用“色阶”使暗处更暗，亮处更亮；“亮度/对比度”适当调整亮度，增强对比度；“自然饱和度”提高，使色彩更丰富。复制一个图层，在“滤镜”中选择“其它-高反差保留”，这会保留尖锐的边界，然后选择叠加模式，就可以让边缘更明显。

    Blender

    基本介绍

    观察视图

    打开后会看到这样一个界面：

    • 左侧是建模操作；
    • 下方是时间轴；
    • 右侧是场景信息；

    通过鼠标中键转动视角；Shift + 鼠标中键平移视角

    由于 utools 占用了鼠标中键，因此通过拖动右上角的小坐标轴以及“手部”图标实现转动视角和平移视角。

    物体操作

    左侧中间的 4 个图标可以调整物体。鼠标移动上去就可以看到快捷键

    调整模式

    第一个图标是“调整”图标，它用于选中物体和移动物体等操作。里面还有

    • 框选：可以一次选择多个物体
    • 刷选：选择刷到的物体
    • 套索选择：选择套索范围的物体

    游标模式

    第二个图标是“游标”图标，它是画面中的十字准心。新建的物体将会出现在游标位置。处于游标模式下，可以移动游标位置。

    • 在调整模式下，可以 Shift + 右键移动游标；
    • 使用 Shift + c 恢复游标位置到原点；
    • 使用 Shift + s 调出游标操作，移动选中项到游标或者移动游标到选中项等；

    物体原点

    物体中的黄色点就是物体的原点，一般的变换都是围绕原点进行。当我们需要进行特殊的操作，例如将立方体的底部放在坐标原点，这时候就需要移动物体原点的位置。

    在右上角选择“仅影响原点”，

    • 这时候进行 g 平移操作就会只移动物体原点，切换到 y 视图，按住 Ctrl 键移动原点吸附到立方体底部，然后取消勾选“仅影响原点”。最后 Alt + g 让物体原点回到坐标原点，那么立方体底部就移动到了原点位置。
    • 如果需要将原点移动到坐标原点，直接 Alt + g 即可，然后取消勾选“仅影响原点”。

    或者直接右键选择“设置原点”来移动原点位置。

    轴心点

    默认轴心点事物体原点，即质心点。当选择多个物体时，

    • 边界框中心：用一个最小的方框框住所有物体，取这个方框的中心
    • 3D 游标：围绕游标变换
    • 质心点：所有物体的质心
    • 各自的原点：围绕各自原点变换
    • 活动元素：围绕最后选中的物体变换

    局部坐标系

    在操作时需要区分局部坐标系和全局坐标系。在全局坐标系下，按 g + x 会沿着全局坐标的 x 轴平移；按 g + x + x 则切换到局部坐标的 x 轴进行平移；其余坐标轴同理。反之，在局部坐标轴下，按 g + x 会沿着局部坐标的 x 轴平移；按 g + x + x 则切换到全局坐标的 x 轴进行平移；其余坐标轴同理。

    应用变换

    这是一个非常重要的部分！！！尽量在编辑模式下进行缩放操作！在物体模式下进行的缩放操作，最好用 Ctrl + a 应用缩放。

    按 n 键查看条目，会发现“变换”部分给出了对物体施加的变换。这意味着这些变换没有修改物体本身的数据，是在物体数据的基础上进行变换。而使用 Ctrl + a 应用变换后，变换将会应用到物体数据，此时物体本身发生了变化，而所有变换都被清空。

    创建集合

    选中物体后，可以按 m 创建集合或者移动到指定集合

    渲染器

    默认提供两种渲染引擎

    快捷键

    需要注意设置输入法为英文，否则一些快捷键会无效。

    Alt 鼠标中键切换视角

    保存图像

    F12 渲染完成后，Alt + s 保存渲染图像。

    当视频编辑区有序列时，将无法渲染图像，自然也不能保存。

    插件

    在偏好设置中搜索插件并勾选即可使用

    插件 作用 插件 作用
    LoopTools 编辑环形网格 Node Wrangler 节点快捷操作
    导入图像为平面 导入图像成为平面模型

    网格建模

    基本控制

    为了方便切换，我们在“偏好设置”中勾选“拖动时的饼菜单”，这样可以长按 Tab 键移动鼠标调出切换模式的菜单。

    切换模式

    进入编辑模式后，可以通过键盘上方（不是数字键）的 1/2/3 切换点/线/面编辑。或者通过菜单图标切换，图标设置可以同时开启多个选择，按住 Shift 键可以全部选中 3 种模式，这样可以同时选择点、线、面。

    选择技巧

    在选择时，可以长按

    • Shift 键加选
    • Ctrl 键减选；配合数字键盘，Ctrl + + 可以扩大选区；Ctrl + - 可以减小选区

    一些常用的选择方法：

    • 选择一个点/线/面，按住 Ctrl 键再选择另一个点/线/面，会自动选择最短路径上的点/线/面；
    • 选择一个点/线/面，按 l 键选择与当前选中元素相连的所有元素（需要鼠标移动到物体上）；或者直接按 Ctrl + l 选中；
    • 选择一个点/线/面，
      • 按住 Alt 键，鼠标轻微移动到一个方向上双击，就可以选中这个方向上的所有元素；
      • 按住 Ctrl + Alt 键，鼠标轻微移动到一个方向上双击，就可以选中垂直于这个方向上的所有元素；
    • 还可以通过“选择”菜单，进行随机选择。

    对于分段的连续边，使用 Shift + Alt 只会选择其中一段上的边，这时候可以使用“选择锐边”选取连续角度不超过指定度数的一系列边

    纹理介绍

    添加纹理

    在节点编辑器界面 Shift + a 添加图像纹理

    将“颜色”与“基础色”相连，然后可以打开一张纹理图像

    纹理节点有一个输入：“矢量”。因此我们建立“矢量-映射”节点

    在当前节点连接下，默认会使用 UV 纹理坐标。其中“线性”-“Flat”-“重复”控制纹理映射的形式，例如“重复”表示对超出纹理边界的部分重复纹理图像。

    纹理坐标

    要修改纹理坐标的形式，首先将“矢量”对应连接

    而映射节点又有“坐标”输入，因此在“输入”中找到“纹理坐标”节点

    可以选择“法向”、“UV”等输出作为矢量输入，控制纹理映射的形式

    映射形式

    纹理映射形式

    • 生成：以全局坐标系映射，只与坐标原点相关
    • 物体：以物体坐标系映射，与物体原点相关
    • 摄像机：以摄像机坐标系映射
    • 窗口：以窗口视角映射

    物体映射模式可以指定坐标轴，例如创建一个空坐标轴，吸取这个坐标轴，就可以通过对这个坐标轴变换来修改映射

    颜色渐变

    创建颜色渐变、分离 XYZ 节点，然后同样创建映射节点和纹理坐标节点

    将它们组合在一起：我们将映射矢量作为纹理渐变的系数，就可以看到纹理随着坐标变化

    可以尝试添加不同的纹理节点来得到不同效果

    物体信息

    使用“输入”中的物体信息节点可以获取物体的各项参数，还可以生成随机值。例如给 3 个模型关联同一种材质，使用物体信息节点随机输出系数，就会对每个模型输出不同的渐变色

    凹凸纹理

    可以通过如下节点连接产生具有凹凸质感的纹理：

    1. 用噪波纹理生成随机颜色
    2. 通过颜色渐变映射到黑白纹理
    3. 黑白纹理通过凹凸节点产生法向高度
    4. 输出法向控制凹凸程度

    法向贴图

    计算机中常用像素颜色代表了法向坐标，即 \((x,y,z)=(r,g,b)\)，而法向贴图就是将法向记录下来的一个方式

    通过指定纹理图像、糙度图像、法向贴图（法向贴图直接连接法向，不连接高度）来实现凹凸纹理。然后通过置换修改器和表面细分修改器进行形变，增加实际的凹凸细节。

    在旋转菜单中设置随机旋转和相位，还可以指定坐标轴向

    确保渲染物体初始局部坐标 z 轴向上，然后全局旋转到 x 方向，这样就可以确保设定坐标轴向为全局 Z 时，生成的物体都向上。

    火焰粒子

    顶点组

    在编辑模式下，选中火焰的上半部分，在“数据”菜单的顶点组处，点击加号创建顶点组，点击“指定”即可分组。如果要选中组，点击“选择”即可选中其中的顶点。

    在左上角切换进入权重模式，然后选择“权重-光滑”来平滑化顶点权重，增加迭代次数实现平滑化，这样可以影响置换修改器的作用效果

    然后在置换修改器中指定顶点组

    创建一个空物体，置换修改器的坐标改为“物体”，吸取空物体，就可以通过空物体的移动来控制形状

    运动帧

    要实现运动效果，可以选中空物体，将 z 轴变换设为 #frame/10，这样按空格播放后，物体就会将帧数作为 z 轴参数

    在火焰中间放置一个圆盘，关闭力场权重中的重力，让粒子向上移动

    世界光

    即使在没有灯光的情况下，场景也存在环境光。调节“强度/力度”为零，就可以修改世界光的强度

    进入着色器编辑器，调整为世界环境，即可查看和编辑世界环境中的光照、材质等信息

    环境布光

    主要有 4 种环境打光方式：

    • 使用渐变纹理节点
    • 使用背景节点
    • 使用天空纹理节点
    • 使用环境纹理节点

    最常用的是背景节点和环境纹理节点。

    天空纹理

    在 Cycles 渲染下，可以通过天空纹理显示出太阳（将空气拉满），得到自然的打光

    环境纹理

    使用 HDR 贴图可以直接得到环境光照。在 Blender 安装目录下可以找到内置的环境贴图

    建立“环境纹理”节点，导入环境贴图

    如果不希望背景干扰，可以勾选渲染菜单中的“胶片-透明”选项

    场景布光

    定基调

    根据实际场景选择环境光，例如室内环境使用对应的 HDR 贴图。借助映射节点，将贴图旋转，让物体位于窗户附近。布置完成后，勾选“胶片-透明”选项，取消背景渲染

    三点布光

    使用主光、辅助光、轮廓光进行三点布光。

    • 主光负责照亮主体
    • 辅助光负责照亮暗部
    • 轮廓光一般从背后照射，负责凸显轮廓

    主光一般亮度最高，辅助光和轮廓光较弱，主要照亮阴影。为了防止轮廓光产生的逆光投出阴影，可以将其“阴影”选项取消勾选。

    阴影效果

    要实现灯光透过指定纹理投影的效果，可以利用“透明 BSDF”节点。指定纹理为噪波纹理，将它作为系数，形成蒙版混合效果；分别为原理 BSDF 和透明 BSDF 指定颜色，然后混合，这样就会根据噪波纹理进行透射：黑色纹理部分不会透射，而白色纹理部分会透光。

    需要在 Cycles 渲染下才能预览渲染效果。

    导入素材

    Blender 有两种导入 blender 素材的方式：

    • 关联：导入素材，但是不能修改。好处是原 blender 文件中的素材修改时，当前关联的素材也会修改（需要“文件-重新加载”，记得先保存）
    • 追加：导入素材，可以修改

    导入方式是双击 .blend 文件，选择内部的物件或集合中的物件，然后导入。

    如果需要导入其它格式的文件，则使用“导入”选项。

    动画

    帧率

    一般帧率设为偶数，因为帧数只能是整数，这样方便精确地插入半秒。

    关键帧

    在物体模式下，选中物体 i 键选择插入的关键帧，此时变换菜单中的“位置”会变成黄色

    移动物体后，在两个位置不同帧处插入关键帧，现在就可以播放看到移动效果。将左侧菜单展开，就可以看到每个属性对应的关键帧

    可以选中关键帧，使用类似于物体模式的操作方式：

    • G 键移动位置
    • S 键缩放长度
    • X 键删除关键帧
    • Shift + d 复制关键帧，这里复制的是物体在关键帧处的位置、缩放、颜色等属性

    还可以点击属性右侧的圆点，表示对这个属性打关键帧；或者移动到这个属性上，i 键打关键帧

    可以开启“自动插帧”，这样移动帧数，然后修改物体属性，就会自动插入一个关键帧

    如果希望只对 x 轴插入关键帧，可以在对应属性上右键选择“插入单项关键帧”

    曲线编辑器

    要调整关键帧之间的过渡曲线，首先切换到曲线编辑器，可以选中控制杆，拖动来调节曲线。这里每条曲线分别对应 x,y,z 方向上的变化

    分组移动

    要同时移动多个物体，比较好的方法是指定一个空物体作为这些物体的父组，这样移动空物体就能够同时移动这些物体

    形态键

    使用形态键来绑定：创建一个基型，然后添加一个新的键，修改物体。现在就可以通过修改“值（明度）“产生连续变形

    修改器

    进入曲线编辑器，选中一条线，n 键弹出菜单，选择“修改器”菜单，然后可以添加“内建函数”

    默认会添加一个正弦函数。由于形态键取值范围为 0-1，因此我们将曲线“值偏移”设为 0.5，振幅设为 0.5，这样就实现了循环播放

    还可以添加循环修改器，将会重复曲线中全部帧之间的运动

    骨架

    在物体模式下创建骨架，进入编辑模式后，可以移动骨架顶点、细分骨架、挤出骨架

    进入姿态模式后，可以摆放姿势

    物体栏中给出了骨架之间的父子级关系，这决定了摆放姿态时的关系

    • 从骨架头部挤出的新骨架会成为子骨架
    • 从骨架尾部挤出的新骨架与当前骨架平级

    在物体模式下，只能移动骨架；在编辑模式下，只能挤出骨架、移动骨架顶点；在姿态模式下，只能摆放姿态。

    只有在姿态模式下，可以为每个骨骼打关键帧。注意每次关键帧属性选择“位置+旋转”。每个骨骼的动作会传递给子骨骼，子骨骼的动作需要再打关键帧。

    绑定骨骼

    以一个立方体为例：先创建一个骨架，进行细分。然后创建立方体，将其 Ctrl + r 切分为足够多份。为了能够观察到骨架，可以勾选右侧“视图显示”中的“在前面”。

    切换到姿态模式，就可以随意调整骨骼使物体变形。如果想取消姿态，选择对应的骨骼，直接右键“清空姿态变换”即可。

    骨骼权重

    左上角切换为权重绘制，可以看到当前骨骼的影响权重。其中蓝色权重为 0，红色权重为 1 。右侧顶点组中选择不同的骨骼，可以看到对应的影响权重。

    连接节点

    节点之间需要将属性对应连接，这需要通过 links 属性。例如再创建两个节点，然后将一个节点的输出链接到另一个节点的输入。

    • 输出属性为 outputs，是一个字典，可以通过下标或者键来获取值（节点）
    • 输入属性为 inputs，是一个字典，可以通过下标或者键来获取值（节点）

    下标的顺序与属性的顺序一般一致。例如

    links = tree.links

# 降噪节点
denoise_node = tree.nodes.new(type="CompositorNodeDenoise")
denoise_node.label = 'Denoise'
links.new(image_node.outputs[0], denoise_node.inputs[0])

# 合成器节点
composite_node = tree.nodes.new(type="CompositorNodeComposite")
composite_node.label = 'Composite'
links.new(denoise_node.outputs[0], composite_node.inputs[0])

    空物体

    使用空物体来确定阵列修改器的移动效果。首先创建一个空物体轴，然后给要操作的物体添加阵列修改器，设为“物体偏移”，吸取空物体轴，这样就可以通过操作空物体来改变移动效果：针对空物体的移动、伸缩、旋转操作会叠加到每个阵列生成物体上。

    由于会叠加变换，注意确保物体本身的所有变换都已经应用。

    使用空物体-立方体可以作为复杂模型的父组：框选复杂模型，再选择空物体，Ctrl + p 设为父组。这样就可以通过空物体方便地移动复杂模型，尤其是带有阵列修改器的模型。

    关联复制

    使用 Alt + d 复制物体，则对原始物体的所有更改都会影响所有复制体。

    曲线吸附

    通过关联复制出一侧的叶片，再 Ctrl + d 复制出另一侧的叶片。选中所有叶片，最后选中杆，Ctrl + p 选择“顶点（基于三点）”。最后为杆添加曲线修改器，则杆会沿着曲线形变，叶片会吸附在杆上形变。

    注意调整路径曲线时，将原点移动到坐标原点。不要调整过大，防止出现变形。如果调整过程中，叶片整体突然转动到另一个平面，可以切换到该平面调整，使其恢复到正视图。（在正视图左右扭动时叶片不会脱离杆，而前后扭动可能导致分离）

    一定要确保叶片自身的旋转、缩放变换都已经应用！！！

    镜像范围

    使用镜像修改器时，有可能建模时一侧的点与镜像后的部分相互覆盖，此时可以开启右侧菜单中的 "Clipping" 选项，限制移动范围

    滑动

    使用 g 移动点时，不容易控制点的移动方向。使用 g + g 进入滑动状态，会沿着所在边的路径移动点

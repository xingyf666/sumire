    Word

    基本介绍

    交叉引用

    参考文献通过编号排序，如果中间插入新的参考文献，只需要选中引用部分，右键“更新域”即可。如果需要插入多个文献，可以选择插入的几个引用，右键“切换域代码”，会看到如下形式的代码

    { REF _Ref192777101 \r \h }{ REF _Ref192777101 \r \h }

    在 \r \h 和 } 之间插入 \#"[0" 表示起始，插入 \#"0]" 表示终止，右键“更新域”即可合并引用

    { REF _Ref192777101 \r \h \#"[0" }{ REF _Ref192777101 \r \h \#"0]" }

    更新问题

    如果右键没有“更新域”，可以全选后使用快捷键 F9 更新。

    插入图片

    分图排版

    插入图片后选中图片，点击“插入-表格-文本转换成表格”来插入多列图片；选中图片，点击“引用-插入题注”来创建标注和说明。

    插入时可以新建题注标签。如果发现交叉引用找不到标签，可以重新创建一下对应的题注标签。

    完成后选中表格，设置样式为“无框线”即可隐藏外部线条

    如果要为多图设置子题注，然后为所有图设置总题注，可以直接在图所在的列表行下方插入一行用来设置子题注，然后为整个列表插入题注。

    题注格式

    在“开始-样式”右下角箭头展开找到“题注”，右键编辑修改题注格式。

    表格边框

    选中表格，右键“边框样式”，选择想要的样式以后，可以用边框刷直接画边框。

    UnicodeMath

    介绍

    UnicodeMath 是 Word 自带的公式编辑器，非常轻量易用。

    基本操作

    创建公式

    Word 公式在“插入-公式”按钮处可以新建公式，更便捷的方法是使用快捷键 Alt + = 调出。

    可以直接在公式编辑器中输入字符、计算符甚至 LaTex 字符，只需要紧跟着输入空格就可以触发渲染

    通过 Ctrl + z 取消渲染，因此如果不想自动渲染，可以在每次自动渲染时撤销一次，就只会撤销渲染效果。

    分数

    使用 / 分隔分子和分母，完成后按空格就会自动转换为标准形式

    如果希望保留 b/c 形式，可以输入 b\/c，即使用反斜杠转义。使用 /= 还可以输入不等号，并且 / 配合各种 LaTex 字符可以为字符添加斜杠

    上下标

    使用类似于 LaTex 的形式使用上下标，区别在于使用 () 而非 {} 包含需要上标或下标的内容

    定界符

    使用 {\close 声明左定界符，\open} 声明右定界符

    注意输入次序

    • 左定界符 {\close + space + space
    • 右定界符 \open + space + space + } + space

    看到虚线框后，在其中输入内容然后回车可以换行。

    另一些定界符，例如 |,|| 通过 \vert, \Vert + space 输入

    括号定界符在输入完成后，按空格可以自动匹配内容的高度

    向上/向下

    使用 \below 可以在前面字符的下方添加内容

    输入次序为 lim + space + \below + space + (n->\infty) + space 。类似地，使用 \above 在上方添加内容。

    使用 \rect 可用方框框住任何表达式。使用 \overbar, \underbar 在表达式上下方添加横线

    输入次序为 \rect + space + space，其余两个类似。

    根式

    指令 \sqrt, \cbrt, \qdrt 分别用于生成二阶、三阶、四阶根式。任意阶根式使用 \sqrt(n&a) 输入

    音标符号

    使用 a\hat + space + space 生成带有帽子的符号；使用 b\bar + space + space 生成带有上划线的符号。

    占位符

    使用占位符调整间距

    多行公式

    方程组

    使用 \eqarray 创建方程数组，输入 \eqarray + space 得到单行虚方框，输入后回车换行；如果想要对齐方程，使用 & 表示对齐位置，@ 表示换行，所有方程用 () 包围后按空格渲染

    矩阵

    使用 \matrix 创建矩阵组，用 & 表示新增一列，@ 表示新增一行

    还可以使用定界符包围或者使用 \pmatrix 生成带有小括号的矩阵。

    公式编号

    在公式结尾添加 #(编号) 然后回车得到公式编号

    作用范围

    被灰色所包围的区域是当前指令的作用范围，需要考虑到指令之间的嵌套关系

    MathType

    介绍

    与 Word 兼容的公式编辑器，使用管理员权限安装 MathType 和 MathTypePJ 后可以在 Word 选项卡中看到

    基本操作

    创建公式

    快捷键 Ctrl + Alt + Q 插入内联公式。将会弹出一个编辑窗口

    • 在“大小”中设置公式的整体字号、上下标字号
    • 在“样式”中设置字母字体（公式、文本、希腊字母）。只有在文本格式下可以输入空格。

    模板公式

    运算符通过上方的预制模板获得比较方便

    快捷键

    常用快捷键

    按键 作用 按键 作用
    Ctrl + h 上标 high Ctrl + l 下标 low
    Ctrl + j 上下标 Ctrl + 9/0 小括号
    Ctrl + [ 中括号 Ctrl + Shift + [ 大括号
    Ctrl + r 根式 Ctrl + f 分式
    Ctrl + / 斜杠分式

    使用快捷键创建的括号可以换行变成向量

    巨算符

    使用 Ctrl + T + S 创建求和符号，Ctrl + T + P 创建乘积符号

    使用 Tab 在不同的虚线框中跳转。

    希腊字母

    使用 Ctrl + g 开头再加字母得到希腊字母

    显示嵌套

    可以选择“视图-显示嵌套”来查看当前公式中每一项的嵌套关系

    保存公式

    对于一些较为复杂的公式，可以将其拖放到最下面的公式栏中，以后直接点击调用

    插入方式

    MathType 有四种插入方式

    1. 内联：不换行，用于在正文中插入字符 Ctrl + Alt + Q
    2. 显示：单行居中无编号 Alt + Q
    3. 右编号：单行居中右编号 Alt + Shift + Q
    4. 左编号

    公式章节编号

    要按照章节为公式编号，选择“插入下一章”或者“插入分隔符”，在当前位置插入一个标记，后续的公式编号将以当前标记为准

    最好将分隔符插入在每一章第一段正文的后面，如果插入在标题后面，可能会影响目录的生成。

    插入分隔符后，如果没有反应，可以选择“插入编号-更新”来刷新编号

    要查看已经插入的章节分隔符，可以在“浏览”选项卡中，切换到“章/节”，然后点击"上一个/下一个"，这样就可以看到所有章节分隔符

    要隐藏分隔符，选择样式中的 MTEquationSection 字体，右键“选择所有实例”

    然后右键“修改”，选择最下面的“格式-字体”

    最后勾选“隐藏”后确定即可。

    不按章节编号

    选择“插入编号-格式化”设置编号格式

    交叉引用

    选择“MathType-插入引用”，然后双击想要引用的公式编号即可引用。

    调整行距

    插入公式后，行距可能发生变化，这时候选择“布局”，点击“页面设置”右下角的按钮

    将“文档网格”选项卡下的“网格”设为“无网格”即可。

    对齐方式

    公式对齐

    使用 Ctrl + ; 在当前位置插入一个对齐符号

    编号对齐

    在插入编号后，可能需要让编号底部对齐。只需要在公式中选择“格式-底部对齐”即可。

    粘贴对齐

    从其它位置复制粘贴过来的公式可能不会正确对齐。可以使用"切换 Tex"操作两次得到正确的对齐；或者右键“段落-中文版式-文本对齐方式”设为居中。

    截图识别

    使用在线公式识别网站识别公式为 LaTex 格式，然后直接将 LaTex 文本复制到 MathType 窗口，可以直接转换为 MathType 公式。

    修改字体

    字体大小

    要修改公式字体大小，双击公式进入编辑器。选择“大小-定义”，设置好不同符号的字号确定即可。

    如果想要批量调整整篇文档中的公式大小，在设置完当前公式后，选择“预置-公式预置-保存到文件”，将当前公式的格式保存下来。可以将其存放在 MathType 安装目录下的 Preferences 下

    然后选择“格式化公式”，使用 MathType 预置文件格式化整篇文档

    编号字体

    在样式中找到 MTDisplayEquation 样式，右键“修改”，选择“格式-字体”进行修改即可。

    公式转换

    LaTex 与 MathType

    使用 Alt + \ 在 LaTex 与 MathType 之间转换，要求 LaTex 公式被 $ 包围

    复制到公式编辑器中时，在“预置-剪切和复制预置”中选择“MathML 或 Tex”，这样从公式编辑器复制到外边会得到 LaTex 格式

    打开“预置-工作区预置”，勾选“允许从键盘输入 Tex 语言”，就可以输入 LaTex 语法，输入完成后回车转换为 MathType 格式

    LaTex 与 UnicodeMath

    输入 UnicodeMath 公式后，可以在“公式”选项卡切换到“LaTex”选项，然后在转换中选择“线性”，即可得到 LaTex 格式；选择“专业”，即可得到渲染后的效果

    UnicodeMath 与 MathType

    输入 UnicodeMath 公式后，在“MathType”选项卡找到“转换公式”，勾选需要的转换类型执行转换

    在公式编辑器中选择“预置-剪切和复制预置”，切换到“MathML 2.0 (no namespace)”

    然后复制公式到外部，粘贴选择“创建一个 OMML 公式”，就可以得到 UnicodeMath 格式
